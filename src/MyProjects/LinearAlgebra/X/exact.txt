  主な実装:

  - Exact の定義
      - Exact f g := range f = ker g
  - ShortExact の定義
      - 単射 f、全射 g、かつ Exact f g
  - LeftSplit / RightSplit
      - LeftSplit f := ∃ r, r ∘ f = id
      - RightSplit g := ∃ s, g ∘ s = id
  - 分裂補題 split_lemma
      - 仮定: ShortExact f g と RightSplit g
      - 構成: p := id - s ∘ g により ker g への射影 projKer を作り、Exact より range f = ker g を使って f の芯域制限
  fKer : A → ker g が全単射になる。r := (fKer)⁻¹ ∘ projKer で r ∘ f = id を証明。
  - 五項補題（中の射の単射性）five_lemma_injective
      - 上下の列が短完全、左右の縦射 α, γ が単射、可換性 β ∘ f₁ = f₂ ∘ α、γ ∘ g₁ = g₂ ∘ β
      - β x = β y ⇒ β(x − y) = 0 ⇒ γ(g₁(x − y)) = 0 ⇒ g₁(x − y) = 0 ⇒ x − y ∈ im f₁ ⇒ x − y = f₁ a
      - 0 = β f₁ a = f₂(α a) と f₂ の単射性、α の単射性から a = 0 ⇒ x = y
  - 蛇の補題の準備（核の列の正確さ）ker_exact_sequence
      - 仮定: 上段 Exact f₁ g₁、下段 ShortExact f₂ g₂、可換性 β ∘ f₁ = f₂ ∘ α、γ ∘ g₁ = g₂ ∘ β
      - 定義:
      - i₁ := f₁ を ker α → ker β へ芯域制限（codRestrict）
      - i₂ := g₁ を ker β → ker γ へ芯域制限（codRestrict）
  - 証明: range i₁ = ker i₂
      - 包含 ⊆: g₁ ∘ f₁ = 0（Exact）から直ちに
      - 包含 ⊇: y ∈ ker β, i₂ y = 0 ⇒ g₁ y = 0 ⇒ y = f₁ a。さらに 0 = β y = β f₁ a = f₂(α a) と f₂ の単射性より α a =
  0、ゆえに a ∈ ker α、したがって y = i₁ ⟨a, α a = 0⟩
