# "Session 3: Assignments (Basic 5 + Challenge) — Linear Algebra 1"

## Problem Specifications

### **S3\_P01**

1. **Title**: Transpose is linear with respect to addition
2. **Learning Objective**: Using the component definition of transpose, demonstrate in Lean that $(A+B)^{\mathsf T}=A^{\mathsf T}+B^{\mathsf T}$.
3. **Problem Statement**: For any $m,n$ and matrices $A,B\in M_{m\times n}$ over $\mathbb{R}$, prove $(A+B)^{\mathsf T}=A^{\mathsf T}+B^{\mathsf T}$.
4. **Lean Goal**: `(A + B)ᵀ = Aᵀ + Bᵀ`
5. **Hint**: Use `ext i j` for component comparison and `simp [Matrix.transpose, Matrix.add_apply]`.

---

### **S3\_P02**

1. **Title**: Commutativity of transpose and scalar multiplication
2. **Learning Objective**: Verify component-wise that $(c\cdot A)^{\mathsf T}=c\cdot A^{\mathsf T}$.
3. **Problem Statement**: For any real number $c$ and matrix $A\in M_{m\times n}$, show $(cA)^{\mathsf T}=cA^{\mathsf T}$.
4. **Lean Goal**: `(c • A)ᵀ = c • Aᵀ`
5. **Hint**: Use `ext i j; simp [Matrix.transpose, Matrix.smul_apply]`.

---

### **S3\_P03**

1. **Title**: Distributive property of matrix multiplication (left distributive)
2. **Learning Objective**: Demonstrate $A(B+C)=AB+AC$ while maintaining awareness of matrix multiplication definitions and structural consistency.
3. **Problem Statement**: Let $A\in M_{m\times n},\ B,C\in M_{n\times p}$. Prove $A(B+C)=AB+AC$.
4. **Lean Goal**: `A ⬝ (B + C) = A ⬝ B + A ⬝ C`
5. **Hint**: Use `simpa using Matrix.mul_add A B C` (or `ext`→`simp [Matrix.mul_apply, Finset.sum_add_distrib]`).

---

### **S3\_P04**

1. **Title**: Transposition of matrix product
2. **Learning Objective**: Handle $(AB)^{\mathsf T}=B^{\mathsf T}A^{\mathsf T}$ using known lemmas in Lean.
3. **Problem Statement**: For $A\in M_{m\times n},\ B\in M_{n\times p}$, prove $(AB)^{\mathsf T}=B^{\mathsf T}A^{\mathsf T}$.
4. **Lean Goal**: `(A ⬝ B)ᵀ = Bᵀ ⬝ Aᵀ`
5. **Hint**: Use `simpa using Matrix.transpose_mul A B`.

---

### **S3\_P05**

1. **Title**: Additivity of matrix-vector multiplication
2. **Learning Objective**: Using the definition of `mulVec`, demonstrate $A(x+y)=Ax+Ay$.
3. **Problem Statement**: For $A\in M_{m\times n},\ x,y\in\mathbb{R}^n$. Prove `A.mulVec (x + y) = A.mulVec x + A.mulVec y`.
4. **Lean Goal**: `Matrix.mulVec A (x + y) = Matrix.mulVec A x + Matrix.mulVec A y`
5. **Hint**: Use `simpa using Matrix.mulVec_add A x y` (or `ext i; simp [Matrix.mulVec, Finset.sum_add_distrib]`).

---

### **S3\_CH (Challenge)**

1. **Title**: Commutativity of transpose with inner product (adjoint relationship)
2. **Learning Objective**: Derive $\langle A x,\ y\rangle = \langle x,\ A^{\mathsf T} y\rangle$ from matrix, vector sum, product, and transpose properties.
3. **Problem Statement**: For $A\in M_{m\times n}$, $x\in\mathbb{R}^n$, $y\in\mathbb{R}^m$.

   $$
     \langle A x,\ y\rangle = \langle x,\ A^{\mathsf T} y\rangle
   $$

   Prove this equality, where $\langle \cdot,\cdot\rangle$ denotes the standard inner product.
4. **Lean Goal**:
   `Matrix.dotProduct (Matrix.mulVec A x) y = Matrix.dotProduct x (Matrix.mulVec Aᵀ y)`
5. **Hint**: Use `simpa using Matrix.dotProduct_mulVec (A:=A) (x:=x) (y:=y)` (or component-wise proof using `ext` and `Finset` sums).
