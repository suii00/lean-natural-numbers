ブルバキ流数学の世界へようこそ！
以下の5つの基本課題から、興味のあるものを選んでください（Lean での着手ヒントも添えました）：

1. **一般位相論** – 連続写像の美しい性質
   *目標*：連続写像の合成が連続であることを、逆像と開集合の挙動から把握し、Lean では `Continuous.comp` を用いて形式化。
   *Lean最短ヒント*:

   ```lean
   theorem continuous_comp {X Y Z : Type*}
       [TopologicalSpace X] [TopologicalSpace Y] [TopologicalSpace Z]
       {f : X → Y} {g : Y → Z} (hf : Continuous f) (hg : Continuous g) :
       Continuous (g ∘ f) :=
     by
       simpa [Function.comp] using hg.comp hf
   ```

2. **代数構造論** – 群の抽象的調和（第一同型定理）
   *目標*：準同型 ( f:G\to H ) に対し ( G/\ker f \cong \operatorname{Im}(f) ) を構成。核と像という**普遍的**な構造で命題化する。
   *Lean最短ヒント*（数学的道筋）：`ker`→商群 `G ⧸ ker f`→自然準同型の全単射・準同型性→像への同型。
   *Lean補足*：mathlib ではしばしば `GroupHom` の代わりに `MonoidHom`、`GroupIso` の代わりに `MulEquiv`／`Subgroup` を使い、`QuotientGroup.quotientKerEquivRange f` が標準同型です。

3. **順序理論** – 選択公理の深淵（ツォルンの補題）
   *目標*：鎖が上界をもてば極大元が存在することを示す。集合・包含・鎖という**順序の構造**だけで議論する。
   *Lean最短ヒント*：`zorn'` あるいは `zorn` 族の補題を呼び出すため、(i) 鎖の定義、(ii) 各鎖に対する上界の存在を与える形に組み替える。

4. **測度論基礎** – 無限の数学的制御（可測写像の合成）
   *目標*：可測写像の合成が可測であることを、σ-代数の生成と逆像の可測性から把握し、Lean では `Measurable.comp` を用いて形式化。
   *Lean最短ヒント*:

   ```lean
   theorem measurable_comp {Ω₁ Ω₂ Ω₃ : Type*}
       [MeasurableSpace Ω₁] [MeasurableSpace Ω₂] [MeasurableSpace Ω₃]
       {f : Ω₁ → Ω₂} {g : Ω₂ → Ω₃}
       (hf : Measurable f) (hg : Measurable g) :
       Measurable (g ∘ f) :=
     by
       simpa [Function.comp] using hg.comp hf
   ```

5. **実解析** – 実数の完璧な構造（完備性）
   *目標*：実数がコーシー列完備であることを示す。**距離空間**の一般論（コーシー列・極限）を通じて、具体から普遍へ。
   *Lean最短ヒント*：mathlib には `[CompleteSpace ℝ]` が用意されています。教育目的なら「コーシー列→収束列」の証明スケッチを言語化し、Lean では既存インスタンスを用いて `IsComplete` を導く方針が自然です。

## 🧭 提出方法（推奨）

* **提出物**：

  1. Lean コード（`.lean` 抜粋で可）
  2. **ブルバキ的スケッチ**（集合・写像・普遍性の観点で 5–10 行）
* **評価観点**（分析時に用います）

  * **集合論基盤**：逆像・像・商・鎖などの集合操作で本質を掴めているか
  * **構造優先**：特定の要素計算に依存せず、一般構造で証明を組み立てたか
  * **普遍性**：普遍性や初・終対象、商・像の普遍性で特徴付けられているか
  * **美的調和**：定義→補題→主定理の流れが簡潔で反復可能か
