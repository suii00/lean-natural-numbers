## 初回課題：定数関数の微分

### 問題文（高校生向け）
定数関数の微分は常に0になることを証明しましょう。つまり、f(x) = c（cは定数）という関数があるとき、すべての点でf'(x) = 0となることを示します。これは「変化しないものの変化率は0」という直感的な理解をLeanで厳密に証明する課題です。

### 数学的背景
- **高校での学習内容**：数学Ⅱ「微分法」における基本的な微分公式
- **Leanでの表現**：微分可能性と導関数の値を厳密に定義し、極限の概念を用いて証明

### Lean定義
```lean
-- 必要なimport文
import Mathlib.Analysis.Calculus.Deriv.Basic
import Mathlib.Data.Real.Basic

-- 問題定義
theorem const_deriv (c : ℝ) : 
  ∀ x : ℝ, deriv (fun _ : ℝ => c) x = 0 := by
  sorry
```

### 証明のヒント

1. **基本概念の確認**：`deriv`は一点での導関数を表し、定数関数は`fun _ => c`と書きます
2. **適用すべき定理**：`deriv_const`という定理がmathlibにあり、これが直接使えます
3. **証明戦略**：各点`x`に対して、定数関数の微分の定義から0になることを示すか、既存の定理を適用します

### 完全な証明

```lean
theorem const_deriv (c : ℝ) : 
  ∀ x : ℝ, deriv (fun _ : ℝ => c) x = 0 := by
  -- すべての点xについて証明
  intro x
  -- mathlibの定数関数の微分定理を適用
  exact deriv_const x c
  
-- 別解：より詳細なアプローチ
theorem const_deriv_detailed (c : ℝ) : 
  ∀ x : ℝ, deriv (fun _ : ℝ => c) x = 0 := by
  intro x
  -- 定数関数はすべての点で微分可能
  have h : DifferentiableAt ℝ (fun _ : ℝ => c) x := differentiableAt_const
  -- 微分可能な定数関数の導関数は0
  rw [deriv_const]
```

### 大学レベルへの発展

- **実解析での一般化**：バナッハ空間における定数写像の微分（Fréchet微分）
- **多変数への拡張**：多変数関数f(x,y) = cの偏微分と全微分
- **測度論的観点**：ほとんど至るところ定数な関数の弱微分
