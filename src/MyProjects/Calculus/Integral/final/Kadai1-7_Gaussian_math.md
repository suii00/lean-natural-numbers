# 課題1〜7とガウス積分の数学的解説

本メモは、`FinalKadaiPractical.lean` で扱っている各課題の数学的背景を簡潔にまとめたものです。証明は基本的な微分積分学の事実（連続性、微分の公式、微分積分学の基本定理、広義積分の定義と極限）に基づきます。

## 前提・基本事実
- 微分積分学の基本定理（FTC）: 連続な関数 `f` に対し、`F' = f` なら `∫_a^b f = F(b) − F(a)`。
- 連鎖律: `d/dx (g ∘ h) = (g' ∘ h) · h'`。
- 初等関数の導関数: `d/dx e^{x} = e^{x}`，`d/dx (x^n) = n x^{n-1}`（`n` は自然数），`d/dx arctan x = 1/(1+x^2)`。
- 広義積分: 有界区間の積分の極限で定義し、必要に応じて極限計算で評価。

---

## 課題1&2: 具体的置換（連鎖律の応用）
目標: `∫_a^b 2x · e^{x^2} dx = e^{b^2} − e^{a^2}`。

- `F(x) := e^{x^2}` とおくと、連鎖律より `F'(x) = 2x · e^{x^2}`。
- 基本定理より `∫_a^b 2x · e^{x^2} = F(b) − F(a) = e^{b^2} − e^{a^2}`。

---

## 課題3: arctan の導関数の積分
目標: `∫_a^b 1/(1+x^2) dx = arctan b − arctan a`。

- 初等事実 `d/dx arctan x = 1/(1+x^2)` を用いる。
- `F(x) := arctan x` として基本定理を適用すれば結論。

---

## 課題4: 部分分数分解による積分（方針）
- 有理関数の積分は、分母の因数分解（実数範囲で一次・二次因子）に応じて係数を決め、一次因子に対しては対数、既約二次因子に対しては `arctan` 型や線形置換に帰着する。
- 注意点:
  - 分母が 0 にならない区間で議論する（定義域の管理）。
  - 分解形での係数決定（連立方程式）と、原始関数の組み立て。

---

## 課題5: 曲線間の面積（`y = x` と `y = x^2`、区間 `[0,1]`）
目標: `∫_0^1 (x − x^2) dx = 1/6`。

- 区間 `[0,1]` では `x ≥ x^2` なので、上関数−下関数で面積が表される。
- `∫_0^1 x dx = 1/2`，`∫_0^1 x^2 dx = 1/3` より、差は `1/2 − 1/3 = 1/6`。

---

## 課題6: 回転体の体積（`y = √x` を x 軸回り、`[0,1]`）
目標: `V = π ∫_0^1 (√x)^2 dx = π ∫_0^1 x dx = π/2`。

- 回転体の体積（ワッシャー法）: 半径 `y(x)` の円の面積 `π y(x)^2` を x で積分。
- ここでは `y(x) = √x` なので `y(x)^2 = x`。よって `V = π ∫_0^1 x dx = π/2`。

---

## 課題7: 広義積分 `∫_1^R 1/x^2 dx` の極限（`R → ∞`）
目標: `lim_{R→∞} ∫_1^R 1/x^2 dx = 1`。

- 原始関数: `d/dx (−1/x) = 1/x^2`（`x ≠ 0`）。
- よって `∫_1^R 1/x^2 dx = (−1/R) − (−1/1) = 1 − 1/R`。
- `R → ∞` で `1/R → 0`，したがって極限は `1`。

---

## ガウス積分の準備（`∫_{−R}^{R} e^{−x^2} dx` の性質）
ここでは R > 0 とする。

1) 非負性・正値性
- `e^{−x^2} > 0` なので、任意の区間で積分は非負。
- 厳密な正値性（`> 0`）は、`[0, r]`（`r := min(R,1)`）での下界 `e^{−x^2} ≥ e^{−1}` から
  `∫_0^r e^{−x^2} dx ≥ r · e^{−1} > 0` を得て、包含により `∫_{−R}^{R} e^{−x^2} > 0`。

2) 上界（簡易）：`≤ 2R`
- `−x^2 ≤ 0` より `e^{−x^2} ≤ 1`。したがって
  `∫_{−R}^{R} e^{−x^2} dx ≤ ∫_{−R}^{R} 1 dx = 2R`。

3) 上界（強い評価）：`≤ √π < 2√π`
- 古典的事実: 全実線でのガウス積分 `∫_{ℝ} e^{−x^2} dx = √π`。
- 被積分関数が非負のため、区間積分は全実線積分を超えない:
  `∫_{−R}^{R} e^{−x^2} dx ≤ ∫_{ℝ} e^{−x^2} dx = √π`。
- したがって自明に `∫_{−R}^{R} e^{−x^2} dx < 2√π`（`√π < 2√π`）。
- 参考: `e^{−x^2}` は偶関数なので、`∫_{−R}^{R} e^{−x^2} dx = 2 ∫_0^R e^{−x^2} dx` も便利。

---

## 備考（測度論的視点）
- 非負関数の単調性: `0 ≤ f ≤ g` なら `∫ f ≤ ∫ g`。
- 区間拡大の単調性（`f ≥ 0`）: 区間が広いほど積分値は大きくなる。
- 広義積分はフィルタ極限（`R → ∞`）で表現し、計算結果と一致する形を作って極限を取るのが堅牢。

以上。
