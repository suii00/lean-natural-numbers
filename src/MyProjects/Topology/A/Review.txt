## 1) 何が入ったか（要点のマップ）

* **積位相の普遍性（射影 π₁, π₂）**
  「`h : Z → X×Y` が連続 ⇔ `π₁ ∘ h` と `π₂ ∘ h` がどちらも連続」が核。PDFの定理 1・2 に対応し、Lean 側では
  `continuous_iff_proj_continuous` とその図式版（`product_diagram_commutes`）で実装済みです。証明は `continuous_fst/snd` と成分からペアを作る `prodMk` で閉じています。&#x20;

* **位相環における連続準同型の構造補題**
  環演算の連続性と合成で連続性が持ち上がる（例：`r ↦ f(r+r)`、`(p₁,p₂) ↦ f(p₁*p₂)`）。これは代数構造と位相の整合を端的に示す部分です。Lean では `topological_ring_hom_continuous` にまとまっています。&#x20;

* **指数法則（コンパクト開位相）と評価の連続性**
  局所コンパクトな $X,Y$ の下で $C(X×Y,Z) \cong C(X,C(Y,Z))$ を `Homeomorph.curry` として実装。`@[simp]` により \*\*β-簡約が“見たまま”\*\*に落ちるので `ext x y; simp` が正攻法になります。評価写像 `ev : C(Y,Z)×Y→Z` は `(Y,Z)` が `LocallyCompactPair` なら連続です。Lean では `exponential_law` と `continuous_eval`、さらに `exponential_law_toEquiv_apply` 等の `simp` 補題が入っています。&#x20;

* **前合成とカリー化の整合性（左右・束ね版/点ごと版）**
  $F\circ(\varphi×\mathrm{id})$ をカリー化すると $(\mathrm{curry}\ F)\circ\varphi$ と一致（第1変数側）。第2変数側も、$\psi:Y'→Y$ に対して **点ごと版**（`@[simp]`）と **束ね版**（`C(X,C(Y',Z))` 上の等式）が整っており、どちらも `ext; simp` で回ります。Lean では `curry_comp` / `curry_comp_right_apply` / `curry_comp_right` が該当。&#x20;

* **ext; simp 自動化**
  β-簡約（curry/uncurry）、`prodMap_apply`、評価の β-簡約が `@[simp]` 化されているので、指数法則まわりはほぼ **`ext x y; simp` 一撃**。&#x20;

### How to use

- **Split continuity into components** with  
  `continuous_iff_proj_continuous : Continuous h ↔ Continuous (π₁ ∘ h) ∧ Continuous (π₂ ∘ h)`.  
  Use `.mp` to decompose a goal, or `.mpr` plus `hf.prodMk hg` to build maps into products.

- **Canonical rewrite pattern** for exponential law:
  ```lean
  ext x y; simp   -- beta rules for curry/uncurry + eval + prodMap_apply
````

* **Right-variable precomposition (bundled)** under `[LocallyCompactPair Y Z]`:

  ```lean
  lemma curry_comp_right (ψ : C(Y', Y)) (F : C(X × Y, Z)) :
    (exponential_law …).toEquiv (F.comp (ContinuousMap.prodMap (ContinuousMap.id _) ψ))
      = (ContinuousMap.compRightContinuousMap ψ).comp ((exponential_law …).toEquiv F)
  ```

````

## 3) 超短い実戦レシピ（貼るだけで効くサンプル）

**分割（A の同値）**：
```lean
-- Goal: Continuous h
have h₁ : Continuous (π₁ ∘ h) := ...
have h₂ : Continuous (π₂ ∘ h) := ...
exact (continuous_iff_proj_continuous (X:=X) (Y:=Y) (Z:=Z) h).mpr ⟨h₁, h₂⟩
````

**指数法則の書き換え**：

```lean
ext x y; simp   -- curry/uncurry・評価・prodMap の β-簡約で自動的に正規形へ
```

**右側前合成（bundled）**：

```lean
-- under [LocallyCompactPair Y Z]
have := curry_comp_right (ψ : C(Y',Y)) (F : C(X×Y,Z))
-- 以降は `ext x y; simp` で常に同型化→β-簡約→終了
```

これらはすべて現在の `TopologyA.lean` の API と `@[simp]` 群に沿っています。
