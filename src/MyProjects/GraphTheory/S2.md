# Session 2 Assignments (Basic 5 + Challenge) — Graph Theory

## Basic Problems

### S2_P01: Vertices and Edge Sets
**Learning Objective**: Understand that edges in a graph are subsets of the vertex set.
**Problem Statement**: For any edge {u,v} in a simple graph G, prove that both u and v belong to the vertex set of G.
**Lean Objective**: `theorem S2_P01 (G : SimpleGraph V) (u v : V) : G.Adj u v → u ∈ (Set.univ : Set V) ∧ v ∈ (Set.univ : Set V)`
**Hints**:
1. Trivially true by the type system
2. Provable using `simp`

### S2_P02: Upper Bound on Number of Edges
**Learning Objective**: Understand the maximum possible number of edges in a graph with n vertices.
**Problem Statement**: Prove that in a simple graph with 3 vertices, the number of edges cannot exceed 3.
**Lean Objective**: `theorem S2_P02 (G : SimpleGraph (Fin 3)) : G.edgeFinset.card ≤ 3`
**Hints**:
1. The complete graph K₃ has the maximum number of edges
2. Use `SimpleGraph.card_edgeFinset_le`

### S2_P03: Properties of an Empty Graph
**Learning Objective**: Understand the properties of a graph with no edges (an empty graph).
**Problem Statement**: Prove that in an empty graph, no two vertices are adjacent.
**Lean Objective**: `theorem S2_P03 (V : Type*) (u v : V) : ¬(⊥ : SimpleGraph V).Adj u v`
**Hints**:
1. `⊥` represents the empty graph
2. Direct proof from the definition

### S2_P04: Subgraph Relationship
**Learning Objective**: Understand the subgraph relationship between graphs from a set-theoretic perspective.
**Problem Statement**: If graph G₁ is a subgraph of G₂, then every edge in G₁ must also be an edge in G₂.
**Lean Objective**: `theorem S2_P04 (G₁ G₂ : SimpleGraph V) (h : G₁ ≤ G₂) (u v : V) : G₁.Adj u v → G₂.Adj u v`
**Hints**:
1. `≤` represents the subgraph relationship
2. Prove by expanding the definition

### S2_P05: Neighbor Set of a Vertex
**Learning Objective**: Understand the concept of a vertex's neighborhood (the set of adjacent vertices).
**Problem Statement**: Prove that if vertex v is in the neighborhood of vertex u in a graph G, then u and v must be adjacent.
**Lean Objective**: `theorem S2_P05 (G : SimpleGraph V) (u v : V) : v ∈ G.neighborSet u → G.Adj u v`
**Hints**:
1. Check the definition of `neighborSet`
2. Correspondence between set membership and adjacency

## Challenge Problems

### S2_CH: Composition of Graph Homomorphisms
**Learning Objective**: Understand that graph homomorphisms form a closed set under composition.
**Problem Statement**: Prove that the composition of two graph homomorphisms g∘f is also a graph homomorphism, where f: G₁→G₂ and g: G₂→G₃.
**Lean Objective**: `theorem S2_CH (G₁ : SimpleGraph V₁) (G₂ : SimpleGraph V₂) (G₃ : SimpleGraph V₃) (f : G₁ →g G₂) (g : G₂ →g G₃) : ∃ h : G₁ →g G₃, ∀ v, h v = g (f v)`
**Hints**:
1. Graph homomorphisms are edge-preserving mappings
2. Use `SimpleGraph.Hom.comp` or construct directly
