# 第4回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 除法定理の基本
**学習目標**: 整数の除法定理を具体例で確認する  
**課題文**: 17 を 5 で割った商が 3、剰余が 2 であることを証明せよ。つまり 17 = 5 * 3 + 2 かつ 0 ≤ 2 < 5 を示せ。  
**Lean 目標**: `theorem div_theorem_example : 17 = 5 * 3 + 2 ∧ 0 ≤ 2 ∧ 2 < 5`  
**ヒント**: 
- `norm_num` で数値計算を検証
- 各条件を `constructor` で分解

### P02: 剰余の非負性
**学習目標**: 自然数の除法における剰余の性質を理解する  
**課題文**: 任意の自然数 n と正の自然数 m に対し、n を m で割った剰余 r は 0 ≤ r < m を満たすことを示せ。  
**Lean 目標**: `theorem remainder_bounds (n m : Nat) (hm : 0 < m) : n % m < m`  
**ヒント**:
- `Nat.mod_lt` を使用
- 剰余の定義的性質

### P03: 除法の一意性
**学習目標**: 商と剰余の一意性を理解する  
**課題文**: a = b * q₁ + r₁ = b * q₂ + r₂ で 0 ≤ r₁ < b かつ 0 ≤ r₂ < b ならば、q₁ = q₂ かつ r₁ = r₂ であることを示せ。（片方向のみ：r₁ = r₂ を示す）  
**Lean 目標**: `theorem division_unique_remainder (a b q₁ q₂ r₁ r₂ : Nat) (hb : 0 < b) (h₁ : a = b * q₁ + r₁) (h₂ : a = b * q₂ + r₂) (hr₁ : r₁ < b) (hr₂ : r₂ < b) : r₁ = r₂`  
**ヒント**:
- `Nat.div_add_mod` の一意性
- または `Nat.mod_eq_of_eq_add_mod` を活用

### P04: ユークリッド互除法の一歩
**学習目標**: gcd の再帰的性質を理解する  
**課題文**: gcd(a, b) = gcd(b, a % b) であることを、a = 21, b = 15 の場合で確認せよ。  
**Lean 目標**: `theorem gcd_step : Nat.gcd 21 15 = Nat.gcd 15 (21 % 15)`  
**ヒント**:
- `Nat.gcd_rec` または直接計算
- `norm_num` で値を確認

### P05: 整除と剰余の関係
**学習目標**: 整除の判定条件を剰余で特徴付ける  
**課題文**: b が a を整除することと、a を b で割った剰余が 0 であることが同値であることを示せ。（片方向のみ）  
**Lean 目標**: `theorem dvd_iff_mod_eq_zero (a b : Nat) (hb : 0 < b) (h : b ∣ a) : a % b = 0`  
**ヒント**:
- `Nat.mod_eq_zero_of_dvd` を使用
- 整除の定義から導出

### チャレンジ問題

### CH: 拡張ユークリッド互除法
**学習目標**: ユークリッド互除法でベズー係数を構成する過程を理解する  
**課題文**: gcd(35, 15) = 5 であり、このとき 5 = 35 * x + 15 * y となる整数 x, y を具体的に求め、その正しさを証明せよ。さらに、|x| ≤ 15/5 かつ |y| ≤ 35/5 を満たすような解が存在することを示せ。  
**Lean 目標**: `theorem extended_euclidean : ∃ (x y : Int), 35 * x + 15 * y = Nat.gcd 35 15 ∧ Int.natAbs x ≤ 3 ∧ Int.natAbs y ≤ 7`  
**ヒント**:
- ユークリッド互除法を逆算：35 = 15 * 2 + 5, 15 = 5 * 3 + 0
- x = 1, y = -2 が一つの解
