# 第8回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 環準同型の像
**学習目標**: 環準同型の像が部分環であることを確認する  
**課題文**: f : ℤ → ℤ/12ℤ を自然な射影とするとき、f の像が ℤ/12ℤ 全体であることを証明せよ。  
**Lean 目標**: `theorem surjective_projection : Function.Surjective (Int.castRingHom (ZMod 12))`  
**ヒント**: 
- 任意の剰余類に対し、その代表元が逆像となる
- `Function.Surjective` の定義を展開

### P02: 第一同型定理の具体例
**学習目標**: 環の第一同型定理を具体的に確認する  
**課題文**: ℤ → ℤ/6ℤ の核が (6) であり、ℤ/(6) ≅ ℤ/6ℤ であることを確認せよ。  
**Lean 目標**: `theorem first_isom_theorem_example : Nonempty (ℤ ⧸ Ideal.span {6} ≃+* ZMod 6)`  
**ヒント**:
- `RingHom.quotientKerEquivOfSurjective` を使用
- または `Ideal.quotEquivOfEq` と既知の同型を組み合わせる

### P03: イデアルの対応定理
**学習目標**: 商環のイデアルと元の環のイデアルの対応を理解する  
**課題文**: I ⊆ J を ℤ のイデアルとするとき、π : ℤ → ℤ/I による J の像が ℤ/I のイデアルであることを示せ。  
**Lean 目標**: `theorem ideal_correspondence (I J : Ideal ℤ) (h : I ≤ J) : ∃ K : Ideal (ℤ ⧸ I), K = Ideal.map (Ideal.Quotient.mk I) J`  
**ヒント**:
- `Ideal.map` がイデアルを返すことを使用
- 存在証明は `use` で構成

### P04: 同型の合成
**学習目標**: 環同型の合成と推移性を理解する  
**課題文**: R ≅ S かつ S ≅ T ならば R ≅ T であることを示せ。  
**Lean 目標**: `theorem isom_transitive (R S T : Type*) [Ring R] [Ring S] [Ring T] (f : R ≃+* S) (g : S ≃+* T) : Nonempty (R ≃+* T)`  
**ヒント**:
- `RingEquiv.trans` で同型を合成
- または `f.trans g` を直接構成

### P05: 中国剰余定理の特殊ケース
**学習目標**: 互いに素な法に対する中国剰余定理を理解する  
**課題文**: ℤ/6ℤ ≅ ℤ/2ℤ × ℤ/3ℤ であることを証明せよ。  
**Lean 目標**: `theorem chinese_remainder_six : Nonempty (ZMod 6 ≃+* ZMod 2 × ZMod 3)`  
**ヒント**:
- `ZMod.prodEquivPi` または `ZMod.ringEquivCongr` を活用
- gcd(2,3) = 1 が鍵

### チャレンジ問題

### CH: 第二同型定理の応用
**学習目標**: 第二同型定理（(I + J)/J ≅ I/(I ∩ J)）を具体例で理解する  
**課題文**: ℤ において、I = (4), J = (6) とするとき、(I + J)/J ≅ I/(I ∩ J) を証明せよ。具体的に、I + J = (2)、I ∩ J = (12) であることを示し、(2)/(6) ≅ (4)/(12) の同型を構成せよ。さらに、両辺が ℤ/3ℤ に同型であることを示せ。  
**Lean 目標**: `theorem second_isom_theorem : let I := Ideal.span {4 : ℤ}; let J := Ideal.span {6 : ℤ}; Ideal.span {2} = I ⊔ J ∧ Ideal.span {12} = I ⊓ J ∧ Nonempty ((I ⊔ J).quotient J ≃+* (I.quotient (I ⊓ J)))`  
**ヒント**:
- gcd(4,6) = 2、lcm(4,6) = 12 の関係を使用
- `Ideal.quotientInfEquivQuotientSup` または手動で同型を構成
