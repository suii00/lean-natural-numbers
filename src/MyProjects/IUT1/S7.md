# 第7回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 直積環の構造
**学習目標**: 環の直積の演算と性質を理解する  
**課題文**: ℤ × ℤ において、(2, 3) · (4, 5) = (8, 15) であることを証明せよ。  
**Lean 目標**: `theorem prod_ring_mul : ((2, 3) : ℤ × ℤ) * (4, 5) = (8, 15)`  
**ヒント**: 
- 直積環の乗法は成分ごと
- `Prod.ext` で成分ごとに等号を示す

### P02: 商環の元の等価性
**学習目標**: 商環における元の等価条件を理解する  
**課題文**: ℤ/(6) において、8 と 2 が同じ剰余類を表すことを証明せよ。  
**Lean 目標**: `theorem quotient_ring_equiv : (8 : ZMod 6) = (2 : ZMod 6)`  
**ヒント**:
- `ZMod` での等号は剰余の等しさ
- `norm_num` または `decide` で計算

### P03: 多項式環の拡張
**学習目標**: 環を係数とする多項式環の構成を理解する  
**課題文**: (ℤ/3ℤ)[X] において、(X + 1)² = X² + 2X + 1 であることを証明せよ。  
**Lean 目標**: `theorem poly_over_zmod : ((X + 1)^2 : Polynomial (ZMod 3)) = X^2 + 2*X + 1`  
**ヒント**:
- `ring` タクティクで展開
- ZMod 3 での 2 + 2 = 1 に注意

### P04: 行列環の構成
**学習目標**: 2×2 行列環の基本演算を理解する  
**課題文**: 2×2 整数行列環において、単位行列 I と任意の行列 A の積が A であることを示せ。  
**Lean 目標**: `theorem matrix_ring_identity (A : Matrix (Fin 2) (Fin 2) ℤ) : 1 * A = A`  
**ヒント**:
- `Matrix.one_mul` を使用
- 行列環の単位元の定義

### P05: 部分環の判定
**学習目標**: 部分集合が部分環をなす条件を確認する  
**課題文**: ℤ[√2] = {a + b√2 | a, b ∈ ℤ} が ℝ の部分環であることを示すため、加法と乗法で閉じていることを確認せよ。（乗法の閉性のみ）  
**Lean 目標**: `theorem subring_closed_mul (a b c d : ℤ) : ∃ (e f : ℤ), (a + b * Real.sqrt 2) * (c + d * Real.sqrt 2) = e + f * Real.sqrt 2`  
**ヒント**:
- 展開して (ac + 2bd) + (ad + bc)√2 の形に整理
- `use ac + 2*b*d, a*d + b*c` で証人を与える

### チャレンジ問題

### CH: 局所化の基本性質
**学習目標**: 環の局所化の構成と普遍性を理解する  
**課題文**: S = {1, 2, 4, 8, ...} を ℤ の 2 の冪の乗法的集合とする。局所化 S⁻¹ℤ において、6/4 = 3/2 であることを示し、さらに 5 が S⁻¹ℤ の単元であることを証明せよ。ここで a/s は a ∈ ℤ, s ∈ S による分数を表す。  
**Lean 目標**: `theorem localization_basic : ∃ (S : Submonoid ℤ) (h : (2 : ℤ) ∈ S), let R := Localization S; ∃ (f : ℤ →+* R), (f 6 / f 4 = f 3 / f 2 : R) ∧ IsUnit (f 5 : R)`  
**ヒント**:
- 2 で生成される部分モノイドを構成
- 局所化での等式は交差積で判定：6 * 2 = 3 * 4
