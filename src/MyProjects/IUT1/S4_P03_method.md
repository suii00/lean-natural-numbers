# S4\_P03 証明手法メモ

## 背景
命題 **S4\_P03** は自然数 $a,b,q_1,q_2,r_1,r_2$ について、$b>0$ の下で
\[
  a = b q_1 + r_1,\quad a = b q_2 + r_2,\quad r_1 < b,\quad r_2 < b
\]
を仮定したとき $r_1 = r_2$ を示す。これは除法定理における剰余の一意性に対応し、ブルバキ的には直積集合 $\mathbb{N}\times\mathbb{N}$ 上の射影と包含から得られる構造的事実として理解できる。

## 証明の全体像
1. **正規化された補題 `remainder_unique` の導入**
   - 先に $q_1 \le q_2$ を仮定した補題を証明し、$q_1,q_2$ の大小関係をケース分けにより吸収する。
   - 補題内では $a = b q_1 + r_1 = b q_2 + r_2$ の情報から差分 $q_2 - q_1$ を抽出し、剰余条件と矛盾する場合を排除する。

2. **差分に対する構造的議論**
   - $b q_2 = b q_1 + b(q_2 - q_1)$ を成立させるため、加法と乗法の可換性・結合性を繰り返し用いて等式を整形する。
   - 射影的には $\mathbb{N}\times\mathbb{N}$ の点 $(q_2,r_2)$ を $(q_1, r_1)$ へ引き戻す写像を構成し、差分を第1成分のみに押し込める操作に相当する。

3. **矛盾による差分の零化**
   - $b(q_2 - q_1) = 0$ を示すため、「もし非零なら $r_1 \ge b$ に達してしまう」という論法を採用する。
   - 具体的には $q_2 - q_1 \ne 0$ を仮定し、$b \le b(q_2 - q_1) \le r_1$ を導き、$r_1 < b$ と矛盾させる。

4. **大小関係からのケース分け**
   - $q_1 \le q_2$ なら補題をそのまま適用し、逆の場合はラベルを入れ替えて同じ補題を用い、最後に対称性で $r_1 = r_2$ を得る。

## Lean コードにおけるポイント
- `have hb0 : b ≠ 0 := ne_of_gt hb` により $b$ の非零性を確保し、`Nat.mul_eq_zero` を活用可能にする。
- `Nat.add_sub_of_le` で $q_2 = q_1 + (q_2 - q_1)$ を正規化し、`Nat.mul_add` と併用して差分を切り出す。
- 矛盾の部分は `by_contra` で $b(q_2 - q_1) ≠ 0$ を仮定し、`Nat.pos_of_ne_zero` や `Nat.mul_le_mul_left` を用いて矛盾を構成する。
- ケース分けは `obtain hle | hle := le_total q₁ q₂` を通じて全順序性を明示し、`remainder_unique` を順方向および反転した等式で適用している。

## 図式的イメージ
$D = \{(q,r) \mid a = bq + r,\ 0 \le r < b\}$ とすると、射影
\[
  \pi_1 : D \to \mathbb{N},\quad \pi_2 : D \to [0,b)
\]
が定まる。剰余の一意性は $\pi_2$ が関数として良定義であること、すなわち $D$ 内の同じ $a$ に対応する元の射影値が一致することを主張している。Lean の補題 `remainder_unique` はこの図式を具現化し、差分 $q_2-q_1$ の零化によって $\pi_2(q_1,r_1) = \pi_2(q_2,r_2)$ を示している。

## まとめ
- 証明は「差分を第1成分へ押し込み、射影の値が変化しないことを示す」構造的発想に基づく。
- Lean の実装は、加法・乗法の正規化と全順序性を組み合わせて、論理的矛盾から差分を消去する手順を忠実に表現している。
- この枠組みはブルバキ的な集合と写像の言語に沿っており、剰余の一意性を射影の一意性として理解できる。
