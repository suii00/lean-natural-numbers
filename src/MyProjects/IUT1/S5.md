# 第5回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 素イデアルの判定条件
**学習目標**: 素イデアルの定義を具体例で確認する  
**課題文**: ℤ において、(5) が素イデアルであることを示せ。つまり、a * b ∈ (5) ならば a ∈ (5) または b ∈ (5) であることを証明せよ。  
**Lean 目標**: `theorem prime_ideal_five : Ideal.IsPrime (Ideal.span {5} : Ideal Int)`  
**ヒント**: 
- `Ideal.span_singleton_prime` を使用
- 5 が素数であることを `Nat.Prime` で示す

### P02: 極大イデアルの判定条件
**学習目標**: 極大イデアルの定義を理解する  
**課題文**: ℤ において、(7) が極大イデアルであることを示せ。  
**Lean 目標**: `theorem maximal_ideal_seven : Ideal.IsMaximal (Ideal.span {7} : Ideal Int)`  
**ヒント**:
- `Int.span_singleton_isMaximal` を使用
- 7 が素数であることが鍵

### P03: 非素イデアルの例
**学習目標**: 合成数が生成するイデアルは素イデアルでないことを理解する  
**課題文**: (6) が素イデアルでないことを示せ。つまり、2 * 3 ∈ (6) だが 2 ∉ (6) かつ 3 ∉ (6) となる例を構成せよ。  
**Lean 目標**: `theorem not_prime_ideal_six : ¬ Ideal.IsPrime (Ideal.span {6} : Ideal Int)`  
**ヒント**:
- 素イデアルの否定を示すには反例を構成
- `push_neg` で条件を展開

### P04: 極大イデアルは素イデアル
**学習目標**: 極大イデアルが素イデアルであることを確認する  
**課題文**: I が ℤ の極大イデアルならば、I は素イデアルであることを示せ。  
**Lean 目標**: `theorem maximal_implies_prime (I : Ideal Int) (h : Ideal.IsMaximal I) : Ideal.IsPrime I`  
**ヒント**:
- `Ideal.IsMaximal.isPrime` を使用
- 可換環での一般的性質

### P05: 零イデアルの素性
**学習目標**: 整域において零イデアルが素イデアルであることを理解する  
**課題文**: ℤ において、零イデアル (0) が素イデアルであることを示せ。  
**Lean 目標**: `theorem zero_prime : Ideal.IsPrime (⊥ : Ideal Int)`  
**ヒント**:
- `Ideal.bot_prime` を使用
- 整域の性質から導かれる

### チャレンジ問題

### CH: ℤ における素イデアルと極大イデアルの一致
**学習目標**: 主イデアル整域における素イデアルと極大イデアルの関係を理解する  
**課題文**: ℤ の非零素イデアル I に対し、I が極大イデアルであることを証明せよ。具体的に、I = (p) で p が素数の場合、I が極大イデアルであることを示せ。さらに、この性質が ℤ が主イデアル整域であることに由来することを説明せよ。  
**Lean 目標**: `theorem prime_iff_maximal_int (p : Nat) (hp : Nat.Prime p) : Ideal.IsMaximal (Ideal.span {(p : Int)} : Ideal Int) ↔ Ideal.IsPrime (Ideal.span {(p : Int)} : Ideal Int)`  
**ヒント**:
- 両方向を示す：極大→素は一般的、素→極大は ℤ の特殊性
- `Int.span_singleton_isMaximal` と主イデアル整域の性質を活用
