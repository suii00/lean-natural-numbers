# 第2回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 整数環の単元
**学習目標**: ℤ の単元（可逆元）が ±1 のみであることを確認する  
**課題文**: 整数 u が ℤ の単元であることと、u = 1 または u = -1 であることが同値であることを示せ。（片方向のみ）  
**Lean 目標**: `theorem int_unit_iff (u : Int) (h : IsUnit u) : u = 1 ∨ u = -1`  
**ヒント**: 
- `Int.isUnit_iff` を使って単元の特徴付けを得る
- `Int.natAbs` を使った議論も有効

### P02: 整数環は整域
**学習目標**: ℤ が整域（零因子を持たない）であることを証明する  
**課題文**: 整数 a, b に対し、a * b = 0 ならば a = 0 または b = 0 であることを示せ。  
**Lean 目標**: `theorem int_no_zero_divisors (a b : Int) (h : a * b = 0) : a = 0 ∨ b = 0`  
**ヒント**:
- `Int.mul_eq_zero` または `mul_eq_zero` を使用
- 整域の定義そのもの

### P03: 素数の基本性質
**学習目標**: 素数が 1 より大きいことを形式的に確認する  
**課題文**: p が素数ならば 2 ≤ p であることを証明せよ。  
**Lean 目標**: `theorem prime_ge_two (p : Nat) (hp : Nat.Prime p) : 2 ≤ p`  
**ヒント**:
- `Nat.Prime.two_le` が直接使える
- または素数の定義から導出

### P04: 最大公約数の存在
**学習目標**: 具体的な整数の最大公約数を計算し、その性質を確認する  
**課題文**: gcd(12, 18) = 6 であることを証明せよ。  
**Lean 目標**: `theorem gcd_twelve_eighteen : Nat.gcd 12 18 = 6`  
**ヒント**:
- `norm_num` で計算可能
- または `Nat.gcd` の定義を展開

### P05: ベズー恒等式の具体例
**学習目標**: 最大公約数の線形結合表現を理解する  
**課題文**: gcd(15, 10) = 5 であり、5 = 15 * 1 + 10 * (-1) と表せることを示せ。  
**Lean 目標**: `theorem bezout_example : ∃ (x y : Int), 15 * x + 10 * y = Nat.gcd 15 10`  
**ヒント**:
- `use 1, -1` で証人を与える
- 計算は `norm_num` または `ring`

### チャレンジ問題

### CH: ユークリッドの補題の特殊ケース
**学習目標**: 素数の整除性に関する基本定理を、具体例で理解する  
**課題文**: 素数 3 が積 a * b を整除するとき、3 が a を整除するか、または 3 が b を整除することを証明せよ。ただし、a, b は自然数とする。  
**Lean 目標**: `theorem euclid_lemma_three (a b : Nat) (h : 3 ∣ a * b) : 3 ∣ a ∨ 3 ∣ b`  
**ヒント**:
- `Nat.Prime.dvd_mul` を使用（3 が素数であることを示す必要あり）
- `norm_num` で 3 の素数性を確認
