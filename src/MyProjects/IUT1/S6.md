# 第6回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 環準同型の基本性質
**学習目標**: 環準同型が単位元を保つことを確認する  
**課題文**: f : R → S が環準同型ならば、f(1) = 1 であることを証明せよ。  
**Lean 目標**: `theorem ring_hom_one (R S : Type*) [Ring R] [Ring S] (f : R →+* S) : f 1 = 1`  
**ヒント**: 
- `RingHom.map_one` を使用
- 環準同型の定義から直接導出

### P02: 環準同型の核
**学習目標**: 環準同型の核がイデアルであることを理解する  
**課題文**: 環準同型 f : ℤ → ℤ/nℤ の核が (n) であることを確認せよ。（n = 6 の場合）  
**Lean 目標**: `theorem kernel_mod_six : RingHom.ker (Int.castRingHom (ZMod 6)) = Ideal.span {6}`  
**ヒント**:
- `RingHom.ker` の定義を展開
- `Int.coe_castRingHom` を活用

### P03: イデアルの逆像
**学習目標**: 環準同型によるイデアルの逆像もイデアルであることを確認する  
**課題文**: f : R → S が環準同型、J が S のイデアルのとき、f⁻¹(J) も R のイデアルであることを示せ。  
**Lean 目標**: `theorem preimage_ideal (R S : Type*) [Ring R] [Ring S] (f : R →+* S) (J : Ideal S) : ∃ I : Ideal R, I = Ideal.comap f J`  
**ヒント**:
- `Ideal.comap` がイデアルを返すことを使用
- 存在証明は `use` で構成

### P04: 多項式環のイデアル
**学習目標**: 多項式環におけるイデアルを理解する  
**課題文**: ℤ[X] において、X で生成される主イデアル (X) に X² - X が含まれることを証明せよ。  
**Lean 目標**: `theorem poly_ideal_membership : (X^2 - X : Polynomial ℤ) ∈ Ideal.span {X}`  
**ヒント**:
- `Ideal.mem_span_singleton` で整除に帰着
- X(X - 1) = X² - X の因数分解を使用

### P05: 単元群の構造
**学習目標**: 一般の環における単元（可逆元）の概念を理解する  
**課題文**: ℤ/5ℤ において、2 が単元であることを示し、その逆元が 3 であることを証明せよ。  
**Lean 目標**: `theorem unit_in_zmod_five : IsUnit (2 : ZMod 5) ∧ (2 : ZMod 5) * 3 = 1`  
**ヒント**:
- `IsUnit` の定義または `Units.isUnit` を使用
- 計算は `norm_num` または `decide`

### チャレンジ問題

### CH: 多項式環における素イデアルと極大イデアルの区別
**学習目標**: ℤ[X] において素イデアルが必ずしも極大イデアルでないことを理解する  
**課題文**: ℤ[X] において、(X) は素イデアルであるが極大イデアルでないことを証明せよ。具体的に、(X) ⊊ (X, 2) ⊊ ℤ[X] となることを示し、これが (X) が極大でないことの証明となることを説明せよ。  
**Lean 目標**: `theorem poly_prime_not_maximal : Ideal.IsPrime (Ideal.span {X} : Ideal (Polynomial ℤ)) ∧ ¬Ideal.IsMaximal (Ideal.span {X} : Ideal (Polynomial ℤ))`  
**ヒント**:
- 商環 ℤ[X]/(X) ≅ ℤ が整域だが体でないことを使用
- または直接 (X) ⊊ (X, p) ⊊ ℤ[X] の chain を構成（p は素数）
