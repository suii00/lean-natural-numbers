# 第1回 課題（基礎5＋チャレンジ） — IUT1

## 基礎問題

### P01: 整数の加法群構造
**学習目標**: 整数環 ℤ の加法が群をなすことを Lean で確認する  
**課題文**: 整数の加法について、結合法則と単位元の存在を証明せよ。  
**Lean 目標**: `theorem int_add_assoc (a b c : Int) : (a + b) + c = a + (b + c)`  
**ヒント**: 
- `Int.add_assoc` が mathlib4 に存在するが、自力で `rfl` や `simp` を試みよ
- 加法の定義から自明に従う

### P02: 整数の環構造
**学習目標**: ℤ が環であることの基本性質（分配法則）を確認する  
**課題文**: 整数において、乗法が加法に対して左分配法則を満たすことを示せ。  
**Lean 目標**: `theorem int_mul_add (a b c : Int) : a * (b + c) = a * b + a * c`  
**ヒント**:
- `ring` タクティクが強力
- または `Int.mul_add` を直接適用

### P03: 整数の順序性質
**学習目標**: ℤ の順序構造と演算の整合性を理解する  
**課題文**: 任意の整数 a, b に対し、a ≤ b ならば a + 1 ≤ b + 1 であることを証明せよ。  
**Lean 目標**: `theorem int_le_add_one (a b : Int) (h : a ≤ b) : a + 1 ≤ b + 1`  
**ヒント**:
- `Int.add_le_add_right` または `linarith` を使用
- 不等式の保存性に注目

### P04: 整数の除法性質（導入）
**学習目標**: 整数における整除関係の基本を把握する  
**課題文**: 2 が偶数 2n を整除することを形式的に示せ（n は任意の整数）。  
**Lean 目標**: `theorem two_divides_even (n : Int) : (2 : Int) ∣ (2 * n)`  
**ヒント**:
- `∣` の定義は `∃ k, b = a * k`
- `use n` で証人を与える

### P05: 整数の絶対値
**学習目標**: 絶対値関数の基本性質（非負性）を証明する  
**課題文**: 任意の整数 a に対し、|a| ≥ 0 であることを示せ。  
**Lean 目標**: `theorem int_abs_nonneg (a : Int) : 0 ≤ Int.natAbs a`  
**ヒント**:
- `Int.natAbs` は自然数を返すので自明に非負
- `Nat.zero_le` または `simp` で解決

### チャレンジ問題

### CH: Well-Ordering Principle の部分的実現
**学習目標**: 自然数の部分集合に最小元が存在する原理を、有限集合で確認する  
**課題文**: 空でない有限な自然数集合には最小元が存在することを、具体的な3要素集合 {2, 5, 3} で示せ。集合の最小元が 2 であることを証明せよ。  
**Lean 目標**: `theorem finite_nat_set_has_min : ∃ m, m ∈ ({2, 5, 3} : Set Nat) ∧ ∀ x ∈ ({2, 5, 3} : Set Nat), m ≤ x`  
**ヒント**:
- `use 2` で最小元の候補を提示
- 各要素との比較は `norm_num` または個別に `simp` で処理
