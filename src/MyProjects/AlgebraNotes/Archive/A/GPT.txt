# A. 🔄 同分野深化：環の第一同型定理 → 加群・ネーターへ

## 1) Ring 第一次同型（**即通る一行**）

```lean
import Mathlib/Algebra/Ring/Quot
import Mathlib/Algebra/Algebra/Subsemiring

noncomputable section
open scoped BigOperators
open Ideal

-- 既存: `Ideal.quotientKerEquivRange φ`（旧名: `Ideal.quotient_ker_equiv_range`）
theorem ring_first_isomorphism {R S : Type*} [Semiring R] [Semiring S]
    (φ : R →+* S) :
    Nonempty (R ⧸ Ideal.ker φ ≃+* φ.range) := by
  classical
  exact ⟨Ideal.quotientKerEquivRange φ⟩
```

### ここからの発展

* **イデアルの役割**：`ker φ` は同型の核、`range φ` は像。`(↥(φ.range))` 上で `Ring` 構造が自動付与。
  便利系：`map_mk`, `ker_le_comap`, `map_comap_subtype` の `simp` セット化。
* **加群の準同型定理**：

  ```lean
  import Mathlib/LinearAlgebra/Quotient

  theorem module_first_iso
      {R M N} [Semiring R] [AddCommMonoid M] [AddCommMonoid N]
      [Module R M] [Module R N] (f : M →ₗ[R] N) :
      Nonempty ((M ⧸ f.ker) ≃ₗ[R] f.range) := by
    classical exact ⟨LinearMap.quotKerEquivRange f⟩
  ```
* **ネーター環の分解**（主として**原始分解/一次分解**）：
  まず `Noetherian R` コンテキストで**有限生成性→昇鎖条件**の `simp/mono` 補題を固め、
  `Associated`, `Irreducible`, `Primary` の**辞書**（`@[simp]` は最小限）を整備。

# B. ↔️ 分野横断：位相群の“第一同型”の位相版

**狙い**：`φ : G →* H` が連続（＋開写像などの仮定）なら
`G/ker φ` と `range φ` が**位相群として**同型（`Homeomorph`）になる。

```lean
import Mathlib/Topology/Algebra/Group
import Mathlib/GroupTheory/QuotientGroup

noncomputable section
open QuotientGroup

variables {G H : Type*}
variables [Group G] [Group H] [TopologicalSpace G] [TopologicalSpace H]
variables [TopologicalGroup G] [TopologicalGroup H]

-- 1) 群準同型から商→像への群準同型（連続性は後で付ける）
def quotToRange (φ : G →* H) :
    (Quotient (φ.ker)) →* (φ.range) :=
by
  -- well-definedness は通常 `QuotientGroup` の universal property で
  -- `QuotientGroup.lift` する。詳細は補題分割で埋める。
  classical
  refine QuotientGroup.lift (φ.ker) ?desc ?well
  · intro g; exact ⟨φ g, ⟨g, rfl⟩⟩
  · intro g h hmem
    -- g⁻¹*h ∈ ker φ を使って像が等しいことを示す（TODO）
    admit

-- 2) 連続性と全単射性を付与
theorem topological_group_isomorphism
    (φ : G →* H) (hφ : Continuous φ) (hop : IsOpenMap φ) :
    ∃ (ψ : (Quotient (φ.ker)) →* (φ.range)),
      Continuous ψ ∧ Function.Bijective ψ := by
  classical
  -- ψ := quotToRange φ
  refine ⟨quotToRange φ, ?cont, ?bij⟩
  · -- 連続性：`Continuous (Quotient.map ...)` / quotient の普遍性＋ hφ
    admit
  · -- 全単射：surj は「像」ゆえ自明、inj は ker で割っていることから
    admit
```

### 仕上げへの鍵

* **Homeomorph へ**：`IsOpenMap φ` があれば**商写像が開写像**になり、連続全単射＋開写像→同相。
  代替：`G` が**コンパクト**・`φ.range` が**Hausdorff**なら連続全単射は同相。
* **商位相**：`QuotientGroup.instTopologicalSpace` と `continuous_quot_lift` 系の利用。
* **開写像定理**：局所コンパクト群など条件で `φ` が開写像になる系を別補題化。

> TODOリスト：`quotToRange` の well-definedness / 連続性 補題を分割し、`@[simp]` を整備。
> 例：`ker_subtype_val`, `range_subtype_val`, `map_mk` などの `simp only` セット。

# C. 🎪 応用統合：圏論的普遍性（**商群＝余等化子**）

**正確な共同等化図**：正規部分群 `N ⊴ G` の商 `G ⧸ N` は
$\pi_1, \pi_2 : G × N ⇉ G$
の**余等化子**（coequalizer）。典型は

$$
\pi_1(g,n)=g,\quad \pi_2(g,n)=g \cdot n .
$$

> ※ご提示の「`N ↪ G` と `π ∘ inclusion`」だけでは**共役**の同一視が足りません。
> `G×N` からの 2本の射にするのが教科書的で、商の普遍性と一致します。

```lean
import Mathlib/CategoryTheory/Category
import Mathlib/CategoryTheory/Limits/Shapes
import Mathlib/GroupTheory/QuotientGroup
import Mathlib/Algebra/Category/Group

open CategoryTheory
open scoped CategoryTheory

namespace GrpQuotAsCoequalizer

-- `GroupCat`（群の圏）は mathlib で `GroupCat`（または `Grp`）として定義済み
-- ここでは構図だけ置きます（詳細は admit を小補題に分解して埋める）

variable {G : Type*} [Group G]
variable (N : Subgroup G) [N.Normal]

-- 対象: G, G × N, G ⧸ N
-- 射: π₁, π₂ : G × N ⟶ G と、商射 q : G ⟶ G ⧸ N
-- 余等化子コーフォーク: q ∘ π₁ = q ∘ π₂

-- スケッチ：`IsColimit` の証明骨格
theorem quotient_as_coequalizer :
  ∃ (Q : Type*) (_ : Group Q) (π : G →* Q),
    True  -- ← 実際は `IsColimit` (Cofork.ofπ ...) を与える
:= by
  classical
  refine ⟨QuotientGroup.Quotient N, inferInstance, QuotientGroup.mk' _, trivial⟩
  -- ここから `π ∘ π₁ = π ∘ π₂`（well-definedness）と
  -- 任意の cofork への一意因子化で `IsColimit` を与える（TODO: 補題分解）
end GrpQuotAsCoequalizer
```

### 一般化の道

* **関手的第一同型**：`Grp` での余等化子＝商、`Ab`（アーベル圏）では**核・余核**を通じて
  いわゆる**像＝余核の核**の因子化（coimage–image factorization）がそのまま抽象化。
