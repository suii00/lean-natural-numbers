## 📝 Phase 0: 微分の基礎 - 3つの視点で見る

### 課題0-1: 定数関数の微分

```lean
import Mathlib.Analysis.Calculus.Deriv.Basic

-- 定数関数の微分は0
example (c : ℝ) : deriv (fun x : ℝ => c) = fun x => 0 := by
  sorry
```

#### 🏫 高校数学の理解
「定数を微分したら0になる。なぜなら変化がないから」

#### 🎓 大学数学の理解
「εδ論法で証明：任意のε>0に対して、|f(x+h)-f(x)|/|h| = 0 < ε」

#### 🔬 形式数学の理解
「型理論において、`deriv`は`ℝ → ℝ`から`ℝ → ℝ`への関数。証明は`deriv_const`という既存の定理を適用するだけ」

**ヒント**: `deriv_const`を探してみましょう。

```lean
-- 答え（まだ見ないでください！）
example (c : ℝ) : deriv (fun x : ℝ => c) = fun x => 0 := by
  ext x
  simp [deriv_const]
```

## 📝 Phase 1: 連続性 - 高校では教わらなかった概念

### 課題1-1: すべての点で連続 vs 区間で連続

```lean
import Mathlib.Topology.Basic
import Mathlib.Topology.Instances.Real

-- 定数関数は連続
example (c : ℝ) : Continuous (fun x : ℝ => c) := by
  sorry
```

#### 🏫 高校数学の「連続」
「グラフが途切れない」「鉛筆を離さずに描ける」

#### 🎓 大学数学の「連続」
**位相空間での定義**: f : X → Y が連続 ⟺ Y の任意の開集合 V に対して、f⁻¹(V) が X の開集合

**εδ定義**: 任意のε>0に対して、あるδ>0が存在して、|x-a|<δ ⟹ |f(x)-f(a)|<ε

#### 🔬 形式数学での扱い
mathlibでは**位相空間の定義**を採用！これが高校数学との最大の違いです。

```lean
-- 答えの構造
example (c : ℝ) : Continuous (fun x : ℝ => c) := by
  -- continuous_const を使う
  sorry
```

---

### 💡 重要な気づき #2: 抽象化のレベル

| レベル | 連続性の定義 | 適用範囲 |
|--------|-------------|---------|
| 高校 | グラフ的直感 | ℝ → ℝ のみ |
| 大学（初級） | εδ論法 | 距離空間 |
| 大学（上級） | 位相的定義 | 任意の位相空間 |
| mathlib | 位相的定義 | 完全に一般的 |

**これがBourbaki的アプローチです！**あなたのStructure Towersと同じ精神ですね。

## 🔍 学習のポイント

### 1. タクティクの使い分け

- `exact`: 既存の定理をそのまま適用
- `apply`: 既存の定理を適用し、前提条件を残す
- `simp`: 簡約化（定義を展開）
- `ext`: 関数の外延性（2つの関数が等しいことを示す）
- `intro`: 仮定を導入
- `rw`: 書き換え

### 2. 定理の探し方

mathlibの命名規則：
- `deriv_const`: 定数の微分
- `deriv_id`: 恒等関数の微分
- `continuous_const`: 定数関数の連続性
- `Continuous.comp`: 連続関数の合成
- `Continuous.add`: 連続関数の和

VS Codeで `Ctrl+Space` を押すと候補が出ます。

### 3. 型を理解する

```lean
#check deriv  -- ℝ → ℝ の関数を ℝ → ℝ に変換
#check Continuous  -- (α → β) → Prop (性質を表す命題)
```
