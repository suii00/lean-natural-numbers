### \=== Session s1: 関数とグラフ・面積 ===

このセッションでは、微積分の基本的な考え方である「面積を求める」という直感を、数式と Lean を用いて厳密に表現する練習をします。最も単純な形である定数関数や一次関数のグラフが描く図形の面積から始め、積分の基本的な性質（加法性、単調性）や対称性（奇関数）へと繋げていきましょう。

-----

#### Q1: 長方形の面積（定数関数の積分）

**【問題文】**
実数 $a, b, c$ があり、$a \\le b$ を満たすとします。区間 $[a, b]$ 上で常に値 $c$ をとる定数関数 $f(x) = c$ を考えます。このとき、$y=f(x)$ のグラフと $x$ 軸、および2直線 $x=a, x=b$ で囲まれた部分の面積は、$c \\times (b-a)$ となることを示してください。

**【Lean定義】**

```lean
import Mathlib
noncomputable section
open Real BigOperators MeasureTheory Filter Topology
open scoped Real BigOperators Topology Interval

namespace Zen.Analysis1.S1

theorem s1_q1 (a b c : ℝ) (h : a ≤ b) :
    ∫ x in a..b, c = c * (b - a) := by
  sorry

end Zen.Analysis1.S1
```

**【ヒント】**

  * **戦略:** これは積分の定義から直接導かれる基本的な公式です。mathlib には、このための補題が用意されています。
  * **補題候補:** `integral_const` という、まさにこのための補題を探してみましょう。
  * **型の要点:** `∫ x in a..b, c` は `∫ x in Icc a b, c` の記法糖衣（notation）です。`a..b` は `Set.Icc a b` を表します。
  * **詰めの一手:** `by simp [integral_const]` のように `simp` タクティクで自動的に証明を試みるか、`rw [integral_const]` で直接書き換えるのが良いでしょう。

**【発展】**
この事実は、より一般の「可測集合」上での「可測関数」の積分（ルベーグ積分）においても、測度有限な集合上の定数関数の積分として `∫ x in E, c = c * volume E` という形で成り立ちます。

#### Q2: 台形の面積（一次関数の積分）

**【問題文】**
実数 $a, b$ があり、$a \\le b$ を満たすとします。一次関数 $f(x) = x$ について、区間 $[a, b]$ での積分値、すなわち $y=x$ のグラフの下の部分の面積（ただし $x$ 軸より下は負の面積と考える）が $\\frac{1}{2}(b^2 - a^2)$ となることを示してください。

**【Lean定義】**

```lean
namespace Zen.Analysis1.S1

theorem s1_q2 (a b : ℝ) (h : a ≤ b) :
    ∫ x in a..b, x = (1/2) * (b^2 - a^2) := by
  sorry

end Zen.Analysis1.S1
```

**【ヒント】**

  * **戦略:** 原始関数を見つけて、微積分学の基本定理を適用するのが定石です。$x$ の原始関数は $\\frac{1}{2}x^2$ です。
  * **補題候補:** `integral_id` という補題がこの計算を直接行います。あるいは、より一般的に `integral_pow` を `n=1` の場合に適用することもできます。
  * **型の要点:** `(1/2)` は `(1 : ℝ) / 2` と解釈されます。`b^2` は `b ^ (2 : ℕ)` です。
  * **詰めの一手:** `rw [integral_id]` で書き換えた後、ゴールは代数的な計算になるので `ring` タクティクが有効です。

**【発展】**
この考え方を一般化すると、$f(x) = x^n$ の積分や、任意の多項式関数の積分を計算できるようになります。これは数値積分の基礎となるシンプソンの公式などにも繋がります。

#### Q3: 面積の足し算（積分区間の加法性）

**【問題文】**
$a \\le c \\le b$ を満たす実数 $a, b, c$ と、区間 $[a, b]$ で積分可能な（連続な）関数 $f$ を考えます。このとき、「$a$ から $b$ までの積分」は「$a$ から $c$ までの積分」と「$c$ から $b$ までの積分」の和に等しいことを示してください。

**【Lean定義】**

```lean
namespace Zen.Analysis1.S1

theorem s1_q3 (a b c : ℝ) (f : ℝ → ℝ) (h_le : a ≤ c ∧ c ≤ b)
    (hf : IntervalIntegrable f a b) :
    ∫ x in a..b, f x = (∫ x in a..c, f x) + (∫ x in c..b, f x) := by
  sorry

end Zen.Analysis1.S1
```

**【ヒント】**

  * **戦略:** 2つの隣接する区間での積分は、それらを合わせた区間での積分と等しくなります。これは積分の非常に重要な性質です。
  * **補題候補:** `integral_add_adjacent_intervals` という補題がこの性質を直接示しています。
  * **型の要点:** `(hf : IntervalIntegrable f a b)` という仮定が重要です。この仮定から、部分区間 $[a, c]$ と $[c, b]$ での可積分性 `IntervalIntegrable f a c` と `IntervalIntegrable f c b` が導かれます（`IntervalIntegrable.mono_set`）。
  * **詰めの一手:** `integral_add_adjacent_intervals` を適用するには、部分区間での可積分性も引数として必要になる場合があるので、`have` を使って示すか、`apply?` で必要な仮定を確認しましょう。

**【発展】**
この性質は測度の σ-加法性 `MeasureTheory.measure_union_eq` の一種と見なせます。これにより、複雑な図形を単純な図形に分割して面積を計算するという強力な道具が得られます。

#### Q4: 奇関数の対称区間での積分

**【問題文】**
すべての実数 $x$ に対して $f(-x) = -f(x)$ を満たす連続関数 $f$ （奇関数）を考えます。このとき、原点に対して対称な区間 $[-a, a]$ （ただし $a \\ge 0$）で $f$ を積分すると、値は $0$ になることを示してください。

**【Lean定義】**

```lean
namespace Zen.Analysis1.S1

theorem s1_q4 (a : ℝ) (f : ℝ → ℝ) (h_cont : ContinuousOn f (Icc (-a) a))
    (h_odd : ∀ x, f (-x) = -f x) :
    ∫ x in -a..a, f x = 0 := by
  sorry

end Zen.Analysis1.S1
```

**【ヒント】**

  * **戦略:** 積分を $[-a, 0]$ と $[0, a]$ の2つに分割します。その後、一方の区間で変数変換 $u = -x$ を行い、もう一方と打ち消し合うことを示します。
  * **補題候補:** mathlib には `integral_odd` という便利な補題があります。これを使うのが早道です。
  * **手動で証明する場合:**
    1.  `integral_add_adjacent_intervals` で `∫ x in -a..0, f x` と `∫ x in 0..a, f x` に分割。
    2.  左の積分で `integral_comp_neg_Icc`（変数変換 $u = -x$）を適用。
    3.  `h_odd` を使って式を整理すると、2つの項が互いに打ち消し合うことがわかります。
  * **詰めの一手:** `rw [integral_odd ...]` の形で補題を適用しましょう。その際、関数が奇関数であること (`h_odd`) や定義域の条件を正しく与える必要があります。

**【発展】**
同様に、偶関数（$g(-x) = g(x)$）の対称区間での積分は、半分 `[0, a]` の区間での積分の2倍になります (`integral_even`)。この性質は、フーリエ級数展開で特定の係数が $0$ になることを示す際に強力な道具となります。

#### Q5: 面積の大小比較（積分の単調性）

**【問題文】**
区間 $[a, b]$ 上で常に $f(x) \\le g(x)$ を満たす2つの連続関数 $f, g$ があるとします。このとき、それぞれの関数の積分値も同じ大小関係 $\\int\_a^b f(x)dx \\le \\int\_a^b g(x)dx$ を満たすことを示してください。

**【Lean定義】**

```lean
namespace Zen.Analysis1.S1

theorem s1_q5 (a b : ℝ) (f g : ℝ → ℝ) (hab : a ≤ b)
    (hf : ContinuousOn f (Icc a b)) (hg : ContinuousOn g (Icc a b))
    (h_le : ∀ x ∈ Icc a b, f x ≤ g x) :
    ∫ x in a..b, f x ≤ ∫ x in a..b, g x := by
  sorry

end Zen.Analysis1.S1
```

**【ヒント】**

  * **戦略:** $h(x) = g(x) - f(x)$ という新しい関数を考えます。仮定より、$h(x) \\ge 0$ です。非負の関数の積分が非負になることを示し、最後に式を移項すれば証明できます。
  * **補題候補:**
      * `integral_mono`: まさにこの命題を示す補題です。
      * `integral_nonneg`: `h(x) ≥ 0` ならば `∫ h(x) dx ≥ 0` であることを示す補題。これと `integral_sub` を組み合わせます。
  * **型の要点:** `ContinuousOn` から `IntervalIntegrable` が導かれる（`ContinuousOn.intervalIntegrable`）ので、積分の存在は保証されています。
  * **詰めの一手:** `integral_mono` を `apply` し、ゴールとして出てくる仮定 `∀ x ∈ Icc a b, f x ≤ g x` を `h_le` で示しましょう。

**【発展】**
この性質は、積分という操作が関数空間上の「順序を保つ」ことを意味します。これは、不等式で与えられる関数の評価（例えば、テイラー展開の剰余項の評価など）で極めて重要になります。

#### Challenge: 積分による三角形の面積の証明

**【問題文】**
底辺の長さが $b$、高さが $h$ の直角三角形の面積が $\\frac{1}{2}bh$ であることはよく知られています。ここで $b\>0, h\>0$ とします。この三角形は、関数 $f(x) = \\frac{h}{b}x$ のグラフと $x$ 軸、そして直線 $x=b$ で囲まれた領域と見なせます。積分を用いて、この領域の面積が $\\frac{1}{2}bh$ となることを証明してください。

**【Lean定義】**

```lean
namespace Zen.Analysis1.S1

theorem s1_challenge (b h : ℝ) (hb : 0 < b) (hh : 0 < h) :
    ∫ x in 0..b, (h / b) * x = 1 / 2 * b * h := by
  sorry

end Zen.Analysis1.S1
```

**【ヒント】**

  * **戦略:** 積分の線形性を利用して、定数部分 $(h/b)$ を積分の外に出し、残った $x$ の積分を計算します。
  * **補題候補:**
      * `integral_const_mul`: `∫ c * f(x) dx = c * ∫ f(x) dx`
      * `integral_id`: `∫ x dx = (1/2) * (b^2 - a^2)`
  * **型の要点:** `h / b` は実数の除算です。
  * **詰めの一手:** `rw [integral_const_mul]` で定数を外に出し、次に `rw [integral_id]` で $x$ の積分を計算します。残ったゴールは代数的な式変形なので、`field_simp` や `ring` タクティクで証明を完了させましょう。`hb` の条件 `0 < b` は、`b` で割る操作が正当であることを保証するために必要です。

**【発展】**
この考え方を拡張すると、回転体の体積（円錐など）を積分で計算する「断面積積分法」に繋がります。また、より一般的な図形の面積は、グリーンの定理を用いた線積分によって計算することもできます。
