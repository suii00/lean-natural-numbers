### Q1（定数関数の面積：長方形の面積）

1. **【問題文】**
   実数 `a b c : ℝ` をとる。区間 `[a,b]` 上の定数関数 `f(x)=c` は連続なので可積分である。
   このとき区間の向きも考慮した**区間積分**が

$$
\int_a^b c\,dx = (b-a)\,c
$$

となることを示せ。特に `a ≤ b` ならば、長方形の面積「底辺×高さ」に一致する。

2. **【Lean定義】**

```lean
theorem s1_q1 (a b c : ℝ) :
    ∫ x in a..b, c = (b - a) * c := by
  simpa using intervalIntegral.integral_const (a := a) (b := b) (c := c)
```

3. **【ヒント】**

* 戦略：区間積分の基本補題 `intervalIntegral.integral_const` をそのまま適用。
* 補題候補：`intervalIntegral.integral_const`（定数関数の区間積分）。
* 型の要点：`∫ x in a..b, c` は「`x` に依らない関数」として解釈される。
* 詰めの一手：`simpa` でゴールと補題の右辺の表示を合わせる（順序が違えば `mul_comm` を使う）。

4. **【発展】**
   `c` をノルム付きベクトル空間の元（スカラー倍許容）に拡張すると、`∫ (const) = (b-a) • c` が成り立つ。

### Q2（一次関数の面積：台形の面積）

1. **【問題文】**
   実数 `a b m n : ℝ` をとる。一次関数 `f(x)=m x + n` は `[a,b]` 上で連続・可積分。

$$
\int_a^b (m x+n)\,dx = m\frac{b^2-a^2}{2} + n(b-a)
$$

を示せ（`a,b` の大小は任意。`a≤b` なら台形の面積公式に一致）。

2. **【Lean定義】**

```lean
theorem s1_q2 (a b m n : ℝ) :
    ∫ x in a..b, m * x + n
      = m * ((b^2 - a^2) / 2) + n * (b - a) := by
  -- 線形性で分解し、定数倍は外に出し、`integral_id`, `integral_const` を適用
  -- `ring` で整理
  sorry
```

3. **【ヒント】**

* 戦略：`∫ (m*x + n) = ∫ (m*x) + ∫ n`（`integral_add`）→ 定数倍を外へ（`integral_const_mul` or `integral_mul_const`）→ `integral_id` と `integral_const` を適用。
* 補題候補：`intervalIntegral.integral_add`, `intervalIntegral.integral_const_mul`, `intervalIntegral.integral_mul_const`, `intervalIntegral.integral_id`, `intervalIntegral.integral_const`, `ring`.
* 型の要点：区間積分の線形性は `IntervalIntegrable` が自動で満たされる連続関数でOK。
* 詰めの一手：代数整理は `ring` / `ring_nf`。

4. **【発展】**
   「微分→原始関数→基本定理」の流れで、`(m x+n)` の原始関数 `m x^2/2 + n x` を使って示す一般戦略を Lean で確立。

### Q3（奇関数の対称区間での積分は 0）

1. **【問題文】**
   `a : ℝ`, `f : ℝ → ℝ`。`f` を奇関数（`f(-x) = - f(x)`）とし、`[-a,a]` で可積分とする。
   このとき

$$
\int_{-a}^{a} f(x)\,dx = 0
$$

を示せ。

* **定義域/正則性/可積分性**：`a ∈ ℝ` 任意、`f` は `[-a,a]` 上可積分、`f` は奇関数。

2. **【Lean定義】**

```lean
theorem s1_q3
    (a : ℝ) (f : ℝ → ℝ)
    (hodd : Function.Odd f)
    (hf : IntervalIntegrable f volume (-a) a) :
    ∫ x in -a..a, f x = 0 := by
  -- 置換 x ↦ -x と奇性を併用。ライブラリ補題（例：`integral_odd`）があれば即座に適用。
  -- ない場合は `change` と `simp [hodd.neg]` を織り交ぜて対称性を証明。
  sorry
```

3. **【ヒント】**

* 戦略：置換 `x ↦ -x` により積分値が `-` 倍に等しいことを示す→ したがって 0。
* 補題候補：`Function.Odd.comp`, `intervalIntegral.integral_comp_neg`（系統）, `intervalIntegral.integral_symm`, `integral_odd`（存在すれば最短）。
* 型の要点：`IntervalIntegrable` の保存は「平行移動・反転」で保たれる補題を使う。
* 詰めの一手：`have : ∫ x in -a..a, f x = - ∫ x in -a..a, f x := ...; linarith`。

4. **【発展】**
   偶関数 `g` に対し `∫_{-a}^{a} g = 2 ∫_{0}^{a} g`。偶・奇分解への道。

### Q4（絶対値の面積：分割と対称性）

1. **【問題文】**

$$
\int_{-1}^{2} |x|\,dx
$$

を計算せよ。`|x|` は連続なので可積分。区間を `[-1,0]` と `[0,2]` に分け、`x<0` で `|x|=-x`，`x≥0` で `|x|=x` を用いる。答えは有理数 `5/2`。

2. **【Lean定義】**

```lean
theorem s1_q4 :
    ∫ x in (-1 : ℝ)..2, |x| = (5 : ℝ) / 2 := by
  -- 0 で分割（`intervalIntegral.integral_interval_split`）
  -- 左は ∫_{-1}^0 (-x)、右は ∫_{0}^2 x
  -- `integral_id`, `integral_const`, `simp [abs_of_nonpos, abs_of_nonneg]` で計算
  sorry
```

3. **【ヒント】**

* 戦略：区間分割→絶対値の定義分岐→一次関数の積分。
* 補題候補：`intervalIntegral.integral_interval_subdivide`/`integral_interval_split`（区間分割）、`abs_of_nonpos`, `abs_of_nonneg`, `intervalIntegral.integral_id`.
* 型の要点：分割点 `0` が区間内にあることの明示が必要。
* 詰めの一手：代数整理は `ring`.

4. **【発展】**
   一般に `A,B ≥ 0` とすると $\int_{-A}^{B} |x| dx = \frac{A^2 + B^2}{2}$。

### Q5（非負関数の積分は非負）

1. **【問題文】**
   `a b : ℝ`（`a ≤ b`）、`f : ℝ → ℝ` が `[a,b]` 上で可積分かつ `0 ≤ f(x)`（全ての `x ∈ [a,b]`）を満たすとする。
   このとき $\displaystyle 0 \le \int_a^b f(x)\,dx$ を示せ。

* **定義域/正則性/可積分性**：`a ≤ b`、`IntervalIntegrable f volume a b`、`∀ x ∈ Icc a b, 0 ≤ f x`。

2. **【Lean定義】**

```lean
theorem s1_q5
    {a b : ℝ} (h_ab : a ≤ b) (f : ℝ → ℝ)
    (h_nonneg : ∀ ⦃x⦄, x ∈ Icc a b → 0 ≤ f x)
    (hf : IntervalIntegrable f volume a b) :
    0 ≤ ∫ x in a..b, f x := by
  -- `intervalIntegral.integral_nonneg_of_nonneg_on` 系の補題で即。
  -- あるいは `by have : (0 : ℝ) ≤ ∫ x in a..b, f x := ...; exact this`
  sorry
```

3. **【ヒント】**

* 戦略：区間上の a.e. 非負性から積分の非負性を得る。
* 補題候補：`intervalIntegral.integral_nonneg_of_nonneg_on`, `ae_of_all`, `measure_mono_null`, `ae_restrict_of_ae`.
* 型の要点：点ごとの条件 `∀ x ∈ Icc a b` を a.e. 版に変換（`ae_all_iff`‑風）して適用。
* 詰めの一手：`simp [h_ab]` で `Icc` と区間の整合をとる。

4. **【発展】**
   `f ≤ g` なら $\int f ≤ \int g$（単調性）。測度論的単調収束定理への入口。

### Challenge（偶・奇分解と対称区間の面積公式）

1. **【問題文】**
   `f : ℝ → ℝ` を `[-a,a]` で可積分とする（`a ≥ 0` は仮定不要だが直感には便利）。
   偶部 `fe(x) := (f(x)+f(-x))/2`，奇部 `fo(x) := (f(x)-f(-x))/2` とおく。

$$
\int_{-a}^{a} f(x)\,dx = 2 \int_{0}^{a} fe(x)\,dx
$$

を示せ（`f = fe + fo`，`fe` は偶・`fo` は奇、Q3 を利用）。

* **定義域/正則性/可積分性**：`IntervalIntegrable f volume (-a) a`。`fe, fo` は線形結合なので同区間で可積分。

2. **【Lean定義】**

```lean
def fe (f : ℝ → ℝ) : ℝ → ℝ := fun x => (f x + f (-x)) / 2
def fo (f : ℝ → ℝ) : ℝ → ℝ := fun x => (f x - f (-x)) / 2

theorem s1_challenge
    (a : ℝ) (f : ℝ → ℝ)
    (hf : IntervalIntegrable f volume (-a) a) :
    ∫ x in -a..a, f x = 2 * ∫ x in (0 : ℝ)..a, fe f x := by
  -- `f = fe f + fo f` を示して `integral_add`。
  -- Q3 を fo に適用してゼロ化。fe は偶なので `∫_{-a}^{a} fe = 2 ∫_{0}^{a} fe`。
  -- `linarith` / `ring` で整理。
  sorry
```

3. **【ヒント】**

* 戦略：`f = fe + fo`，`fe` 偶・`fo` 奇、`∫_{-a}^{a} fo = 0`（Q3）→ 目的式。
* 補題候補：`Function.Even`, `Function.Odd`, `intervalIntegral.integral_add`, `integral_odd`，`integral_even`.
* 型の要点：可積分性は線形性で継承。`simp [fe, fo]` で展開。
* 詰めの一手：最後は右辺の `2 * ∫_0^a` の形に合わせて等式変形。

4. **【発展】**
   対称測度空間での「偶/奇」分解の一般化。フーリエ余弦/正弦展開の入口。

## 【完全解答（教師用）】代表：Q1

* 数学の要点：定数関数の区間積分は「区間長 × 定数」。向き付き区間では `b<a` のとき負になる。
* Lean での最短径：ライブラリ補題 `intervalIntegral.integral_const` を `simpa` でそのまま適用。

```lean
-- （再掲）Q1 の主張を example として書き下し
example (a b c : ℝ) : ∫ x in a..b, c = (b - a) * c := by
  simpa using intervalIntegral.integral_const (a := a) (b := b) (c := c)
```

> 補足：もし右辺を `c * (b - a)` と書いた場合は、`by simpa [mul_comm] using ...` として順序を合わせる。
