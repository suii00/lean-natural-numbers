# 🎓 代数構造論：群の第一同型定理 - 補題分割版

## 📋 メイン定理の分解構造

群の第一同型定理を以下の **7つの基本補題** に分割し、段階的に構築していきます：

```lean
-- メイン定理
theorem first_isomorphism_theorem {G H : Type*} [Group G] [Group H]
    (f : GroupHom G H) :
    Nonempty (GroupIso (G ⧸ f.ker) (Set.range f.toFun))
```

---

## 🔧 補題1：核の正規部分群性

```lean
lemma kernel_normal {G H : Type*} [Group G] [Group H] (f : GroupHom G H) :
    f.ker.Normal
```

**概念的意義**: 準同型写像の核は常に正規部分群となる  
**ブルバキ的視点**: 群準同型の構造が自動的に生成する対称性  
**証明の核心**: `g⁻¹ * ker * g ⊆ ker` の普遍的成立

---

## 🔧 補題2：商群の良定義性

```lean
lemma quotient_group_well_defined {G H : Type*} [Group G] [Group H] 
    (f : GroupHom G H) :
    ∀ (g₁ g₂ : G), g₁ * f.ker = g₂ * f.ker → f g₁ = f g₂
```

**概念的意義**: 同値類上での写像の一意性  
**ブルバキ的視点**: 商構造における代表元独立性  
**証明戦略**: `g₁⁻¹ * g₂ ∈ ker ↔ f(g₁⁻¹ * g₂) = 1`

---

## 🔧 補題3：誘導写像の存在

```lean
lemma induced_map_exists {G H : Type*} [Group G] [Group H] (f : GroupHom G H) :
    ∃ (φ : G ⧸ f.ker → Set.range f.toFun), 
    ∀ g : G, φ (QuotientGroup.mk g) = ⟨f g, Set.mem_range_self g⟩
```

**概念的意義**: 商群から像への自然な写像の構成  
**ブルバキ的視点**: 普遍的性質による写像の一意的決定  
**構造的美**: 交換図式 `G → H` と `G → G/ker → im(f)` の一致

---

## 🔧 補題4：誘導写像の準同型性

```lean
lemma induced_map_is_hom {G H : Type*} [Group G] [Group H] (f : GroupHom G H) :
    let φ := classical_induced_map f
    ∀ (x y : G ⧸ f.ker), φ (x * y) = φ x * φ y
```

**概念的意義**: 商群の演算と像の演算の構造保存  
**ブルバキ的視点**: 群構造の射影による保存  
**技術的要点**: 商群での演算の代表元独立性

---

## 🔧 補題5：誘導写像の単射性

```lean
lemma induced_map_injective {G H : Type*} [Group G] [Group H] (f : GroupHom G H) :
    let φ := classical_induced_map f
    Function.Injective φ
```

**概念的意義**: 異なる同値類は異なる像を持つ  
**ブルバキ的視点**: 商構造の最小性（余分な同一視なし）  
**証明の核心**: `φ(gker) = φ(hker) → f(g) = f(h) → g⁻¹h ∈ ker`

---

## 🔧 補題6：誘導写像の全射性

```lean
lemma induced_map_surjective {G H : Type*} [Group G] [Group H] (f : GroupHom G H) :
    let φ := classical_induced_map f
    Function.Surjective φ
```

**概念的意義**: 像の各元素が商群の元素と対応  
**ブルバキ的視点**: 構造射の範囲の完全一致  
**自明性の美**: 構成により自動的に成立

---

## 🔧 補題7：群同型の構成

```lean
lemma construct_group_iso {G H : Type*} [Group G] [Group H] (f : GroupHom G H) :
    GroupIso (G ⧸ f.ker) (Set.range f.toFun)
```

**概念的意義**: 前6補題の統合による同型写像の完成  
**ブルバキ的視点**: 普遍的構成による必然的同型  
**美的調和**: 準同型 → 核 → 商 → 像 の完璧な循環
