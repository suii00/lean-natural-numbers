- Mordell曲線: Δ = 4·0³ + 27·(-2)² = 108
- Congruent curve: Δ = 4·(-1)³ + 27·0² = -4  
- Rank one: Δ = 4·1³ + 27·1² = 31

すべて ≠ 0 で非特異性確認済み

### 有理点検証

すべての主張された有理点が実際に対応する曲線上にあることを確認：
- (3, 5) ∈ Mordell曲線: 5² = 3³ - 2 = 25 ✓
- (0, 0) ∈ Congruent curve: 0² = 0³ - 0 = 0 ✓
