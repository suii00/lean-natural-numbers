## 🔍 随伴関手とは？

**随伴関手（Adjunction）**は、2つの関手 `F : C ⥤ D` と `U : D ⥤ C` の間の特別な関係です：

```
F ⊣ U  （F は U の左随伴）
```

これは以下のいずれかで特徴づけられます：

### 表現1：Hom集合の同型
```
HomD(F(X), Y) ≅ HomC(X, U(Y))  （自然同型）
```

### 表現2：単位と余単位
- **単位（unit）** `η : 1C ⟶ U ∘ F`
- **余単位（counit）** `ε : F ∘ U ⟶ 1D`

これらは**三角恒等式**を満たします：
```
(F η_X) ; ε_{F(X)} = id_{F(X)}    （左三角恒等式）
η_{U(Y)} ; (U ε_Y) = id_{U(Y)}    （右三角恒等式）
```

## 🎯 視覚的理解

```
         Type u
           |
         Free ↓
           |
    StructureTower  ←─ 普遍性
           |
    forgetCarrier ↓

```

自由構造塔は「最小限の構造を追加」し、忘却は「構造を除去」します。この相補的な関係が随伴です。

/-!

自由構造塔関手と忘却関手の間の随伴関係を形式化する。
これは普遍性の圏論的定式化である。

/-- 最小層を持つ構造塔 -/
structure StructureTowerWithMin where
  carrier : Type u
  Index : Type u
  [indexPreorder : Preorder Index]
  layer : Index → Set carrier
  covering : ∀ x : carrier, ∃ i : Index, x ∈ layer i
  monotone : ∀ {i j : Index}, i ≤ j → layer i ⊆ layer j
  minLayer : carrier → Index
  minLayer_mem : ∀ x, x ∈ layer (minLayer x)
  minLayer_minimal : ∀ {x i}, x ∈ layer i → minLayer x ≤ i

namespace StructureTowerWithMin

instance instIndexPreorder (T : StructureTowerWithMin) : Preorder T.Index :=
  T.indexPreorder

variable {T T' T'' : StructureTowerWithMin}

/-- 構造塔の射 -/
structure Hom (T T' : StructureTowerWithMin) where
  map : T.carrier → T'.carrier
  indexMap : T.Index → T'.Index
  indexMap_mono : ∀ {i j : T.Index}, i ≤ j → indexMap i ≤ indexMap j
  layer_preserving : ∀ (i : T.Index) (x : T.carrier),
    x ∈ T.layer i → map x ∈ T'.layer (indexMap i)
  minLayer_preserving : ∀ x, T'.minLayer (map x) = indexMap (T.minLayer x)

/-- 恒等射 -/
def Hom.id (T : StructureTowerWithMin) : Hom T T where
  map := _root_.id
  indexMap := _root_.id
  indexMap_mono := fun hij => hij
  layer_preserving := fun _ _ hx => hx
  minLayer_preserving := fun _ => rfl

/-- 射の合成 -/
def Hom.comp (g : Hom T' T'') (f : Hom T T') : Hom T T'' where
  map := fun x => g.map (f.map x)
  indexMap := fun i => g.indexMap (f.indexMap i)
  indexMap_mono := fun hij => g.indexMap_mono (f.indexMap_mono hij)
  layer_preserving := fun i x hx =>
    g.layer_preserving (f.indexMap i) (f.map x) (f.layer_preserving i x hx)
  minLayer_preserving := by
    intro x
    calc T''.minLayer (g.map (f.map x))
        = g.indexMap (T'.minLayer (f.map x)) := by rw [g.minLayer_preserving]
      _ = g.indexMap (f.indexMap (T.minLayer x)) := by rw [f.minLayer_preserving]

@[ext]
lemma Hom.ext {f g : Hom T T'} (hmap : f.map = g.map) (hindex : f.indexMap = g.indexMap) :
    f = g := by
  cases f; cases g; cases hmap; cases hindex; rfl

/-- 構造塔は圏をなす -/
instance : CategoryTheory.Category StructureTowerWithMin where
  Hom := Hom
  id := Hom.id
  comp := fun f g => Hom.comp g f
  id_comp := by intros; rfl
  comp_id := by intros; rfl
  assoc := by intros; rfl

open CategoryTheory

/-- 基礎集合への忘却関手 -/
def forgetCarrier : StructureTowerWithMin ⥤ Type u where
  obj := fun T => T.carrier
  map := fun f => f.map
  map_id := by intro T; funext x; rfl
  map_comp := by intro T T' T'' f g; funext x; rfl

/-- 半順序集合から自由構造塔を構成 -/

  carrier := X

  covering := by intro x; use x; exact le_refl x

## 💡 段階的ヒント

<details>
<summary><b>レベル1ヒント: 各エラーの本質</b></summary>

1. **エラー1 (Free関手)**: `Type u`の対象には一般的にPreorderがない！Monotoneの証明が不可能
2. **エラー2 (単位の自然性)**: `Free.map f`の`map`フィールドが`f`であることを示す必要
3. **エラー3 (余単位)**: `minLayer (id x) = minLayer x`を示す（定義から自明）
4. **エラー4 (左三角)**: 2つの射の合成が恒等射であることを`Hom.ext`で示す
5. **エラー5 (右三角)**: 関数の合成が恒等関数であることを示す

</details>

<details>
<summary><b>レベル2ヒント: 根本的な設計問題</b></summary>

**重要な洞察**: `Free`関手の定義には問題があります！

`Type u`から`StructureTowerWithMin`への関手を定義するには、各型`X`に何らかの順序を与える必要があります。しかし、一般の型には順序がありません。

**解決策**: `Free`の定義域を変更する必要があります：

```lean
-- 正しいアプローチ
def Free : Preorder.{u} ⥤ StructureTowerWithMin
```

ここで`Preorder`は半順序を持つ型の圏です。

または、具体的な例で随伴を示す：
```lean
-- Natに特化した例
def FreeNat : Unit ⥤ StructureTowerWithMin
```

</details>

<details>
<summary><b>レベル3ヒント: 修正された定義</b></summary>

問題を修正するため、半順序集合の圏を考えます：

```lean
-- 半順序集合の圏の射
structure PreorderHom (X Y : Type u) [Preorder X] [Preorder Y] where
  toFun : X → Y
  monotone' : Monotone toFun

-- これを使ってFree関手を定義
def Free : Type u ⥤ StructureTowerWithMin where
  obj := fun X => freeStructureTowerMin X
  map := fun {X Y} (f : X → Y) => {
    map := f
    indexMap := f
    indexMap_mono := fun {i j} hij => hij  -- f が単調でないと仮定できない！
    ...
  }
```

**別のアプローチ**: 自然数など具体例で随伴を示す方が教育的かもしれません。
