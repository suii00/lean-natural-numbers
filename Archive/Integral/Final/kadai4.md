## 数学的意味の解説

### 定理の内容
```lean
∫ x in (1:ℝ)..(2:ℝ), 1 / (x * (x + 1)) = Real.log 2 - Real.log (3/2)
```

これは区間 [1, 2] における有理関数 `1/(x(x+1))` の定積分を計算しています。

### 1. 被積分関数の意味

**`1/(x(x+1))`** は有理関数で、以下の特徴があります：
- 分母が `x` と `x+1` の積
- `x = 0` と `x = -1` で特異点（極）を持つ
- 積分区間 [1, 2] では連続で正の値を取る

### 2. 部分分数分解の原理

```
1/(x(x+1)) = 1/x - 1/(x+1)
```

**なぜこうなるか**：
```
1/x - 1/(x+1) = (x+1)/(x(x+1)) - x/(x(x+1))
                = ((x+1) - x)/(x(x+1))
                = 1/(x(x+1))
```

これにより、複雑な分数を単純な分数の差に分解できます。

### 3. 積分の計算過程

```
∫₁² 1/(x(x+1)) dx = ∫₁² (1/x - 1/(x+1)) dx
                   = ∫₁² 1/x dx - ∫₁² 1/(x+1) dx
                   = [log x]₁² - [log(x+1)]₁²
                   = (log 2 - log 1) - (log 3 - log 2)
                   = log 2 - 0 - log 3 + log 2
                   = 2 log 2 - log 3
```

### 4. 結果の変形

```
2 log 2 - log 3 = log 2² - log 3
                 = log 4 - log 3
                 = log(4/3)
                 = log 2 - log(3/2)
```

### 5. 幾何学的解釈

この積分は、曲線 `y = 1/(x(x+1))` と x軸、および直線 `x = 1`, `x = 2` で囲まれた領域の面積を表します。

```
y
|
|    領域の面積 = log(4/3) ≈ 0.288
|    
|  ╱╲
| ╱  ╲___
|╱       ╲___
+-------------→ x
 1         2
```

### 6. 数値的意味

- `log 2 ≈ 0.693`
- `log(3/2) ≈ 0.405`
- `log 2 - log(3/2) ≈ 0.288`

これは比較的小さな面積で、関数が `x = 1` で値 `1/2`、`x = 2` で値 `1/6` を取ることと整合的です。

### 7. 応用例

このタイプの積分は以下の分野で現れます：

- **確率論**: 特定の確率分布の正規化定数
- **物理学**: RC回路の過渡応答
- **経済学**: 割引率が変化する場合の現在価値計算
- **数値解析**: 有理関数近似の誤差評価

### 8. 一般化

この手法は一般的な有理関数
```
P(x)/Q(x)
```
（ここで deg(P) < deg(Q)）に対して適用可能で、Q(x)を因数分解して部分分数分解することで、複雑な積分を単純な対数関数の組み合わせとして表現できます。
