以下は、ガウス積分の正値性・上界（および周辺の課題）を Lean で通すために実務でよく使う mathlib API の要点です。典型的な使い方（簡単なコード断片）も添えます。

Interval Integral

intervalIntegral.integral_eq_sub_of_hasDerivAt: 原始関数で評価する基本定理。
例: have := intervalIntegral.integral_eq_sub_of_hasDerivAt hF hint
intervalIntegral.integral_const: 区間の長さ × 定数。
例: ∫ x in a..b, (c:ℝ) = c * (b - a)
intervalIntegral.integral_add / integral_sub: 線形性。
intervalIntegral.integral_mono_on（または integral_mono_of_nonneg 系）: 点wise比較から積分の比較。
例: have h := intervalIntegral.integral_mono_on (a := -R) (b := R) hI1 hI2 (by intro x hx; exact h_le_one x)
intervalIntegral.integral_pos_of_* 系: a.e. 非負＋a.e. 正＋長さ>0 ⇒ 積分が厳密正（環境差があるため、定数下界で代替可能）
微分（HasDerivAt）

hasDerivAt_pow (n := k): x^k の導関数。
Real.hasDerivAt_arctan: d/dx arctan x = 1/(1+x^2).
HasDerivAt.inv: x ≠ 0 上での逆数の導関数。
補助変換: .const_mul, .mul_const, .exp, .comp など。
例: hx3.const_mul (1/3 : ℝ), hx2.exp
連続性・可積分性

Real.continuous_exp, continuous_neg, (continuous_id).pow k
Continuous.intervalIntegrable, ContinuousOn.intervalIntegrable
例: hcont.intervalIntegrable a b で区間可積分性を確保
点ごとの評価 → 積分評価

Real.exp_pos: exp t > 0（非負・正の確保）
Real.exp_le_one_iff: t ≤ 0 ↔ exp t ≤ 1（上界 ≤ 1 の定式化）
例: have : -x^2 ≤ 0 := by nlinarith; simpa [Real.exp_zero] using (Real.exp_le_one_iff.mpr this)
Real.exp_le_exp.mpr: a ≤ b → exp a ≤ exp b（単調性で下/上界を移す）
正値性の出し方（2通り）

定数下界法（安定）:
[0,1] 上で exp (-x^2) ≥ exp (-1) を示し、∫ ≥ (min R 1 - 0) * exp(-1) > 0
使うAPI: 上の単調性、integral_const、integral_mono_on
a.e. 正を使う法:
intervalIntegral.integral_pos_of_ae_nonneg_of_ae_pos (a := -R) (b := R) ...（環境により名前差あり）
a.e. 正は Filter.eventually_of_forall (λ x, Real.exp_pos _) などで供給
対称性（偶関数）

exp (-x^2) は偶関数: f (-x) = f x
対称区間の簡略化（任意）:
∫ x in -R..R, f x = 2 * ∫ x in 0..R, f x（偶関数向けの補題や integral_comp_neg の組み合わせ）
フィルタ・広義積分（課題7関連）

Filter.eventually_atTop, Filter.Tendsto.congr'
tendsto_inv_atTop_zero, tendsto_const_nhds, .sub, .add などの合成
例: have h_inv0 : Tendsto (fun R => 1 / R) atTop (nhds 0) := by simpa [one_div] using tendsto_inv_atTop_zero
集合・順序の小技

Set.uIcc と Set.Icc の変換: Set.uIcc_of_le h, Set.mem_Icc.mp/.mpr
min 系: min_le_right, le_min_iff, lt_of_le_of_lt
非零・非負: sq_nonneg x, pow_ne_zero, ne_of_gt, lt_of_lt_of_le
タクティクス・補助

代数・評価: ring, nlinarith, linarith, norm_num, field_simp
書換え: simp/simp [one_div, mul_comm, mul_assoc, Real.exp_zero, Set.uIcc_of_le]
これらを組み合わせれば、次の2点が機械的に通せます。

上界: 点wise に exp (-x^2) ≤ 1 を出して integral_mono_on → ∫ ≤ ∫ 1 = 2R
正値: [0, min R 1] で exp (-x^2) ≥ exp(-1) を示し、integral_const と integral_mono_on で > 0
必要なら、あなたの gaussian_bounded の証明をこの API で書き換え・短縮する実装も行います。
