### **🚨 問題1: `deriv_mul` のパターンマッチング失敗**

**原因分析:**
- 積の微分公式 `deriv_mul` は適切な微分可能性の仮定が必要
- 型推論の失敗による引数の不一致
- 暗黙の型パラメータの解決不能

**解決策:**

```lean
-- ❌ 失敗例
have h : deriv (f * g) = deriv f * g + f * deriv g := deriv_mul  -- エラー

-- ✅ 正しい使用法
example (f g : ℝ → ℝ) (x : ℝ) (hf : DifferentiableAt ℝ f x) (hg : DifferentiableAt ℝ g x) :
    deriv (f * g) x = deriv f x * g x + f x * deriv g x :=
  deriv_mul hf hg
```

### **🚨 問題2: `HasDerivAt` API の型適用エラー**

**原因分析:**
- 基底体 𝕜 (NontriviallyNormedField) の型推論失敗
- ベクトル空間構造の暗黙引数の不明確さ
- フィルター引数の混乱

**解決策:**

```lean
-- ❌ 型エラーの例
have h : HasDerivAt f (2*x) x := sorry  -- xが束縛されていない

-- ✅ 正しい型指定
example (x : ℝ) : HasDerivAt (fun t ↦ t^2) (2*x) x := by
  simp [hasDerivAt_pow]

-- ✅ 明示的な型パラメータ
example (f : ℝ → ℝ) (f' x : ℝ) : HasDerivAt f f' x ↔ 
    Tendsto (fun h ↦ (f (x + h) - f x) / h) (𝓝[≠] 0) (𝓝 f') :=
  hasDerivAt_iff_tendsto_slope
```

### **🚨 問題3: Mathlib API の正確な使用法の把握不足**

**主要APIの正しい理解:**

**1. 微分可能性の階層:**
```lean
DifferentiableAt ℝ f x      -- 微分可能
  ↕ (iff)
∃ f', HasDerivAt f f' x     -- 導関数が存在
  ↕ (deriv)  
deriv f x                   -- 導関数値（存在しなければ0）
```

**2. 重要な定理の正しい適用:**
- **積の公式**: `deriv_mul (hf : DifferentiableAt ℝ f x) (hg : DifferentiableAt ℝ g x)`
- **合成の公式**: `HasDerivAt.comp`
- **基本関数**: `hasDerivAt_pow`, `hasDerivAt_sin`, `deriv_exp`

### **💡 実用的な解決パターン**

**パターン1: 型推論エラー対策**

**パターン2: `simp`タクティクの活用**

```lean
  simp [deriv_add, deriv_const_mul, deriv_pow, deriv_const]
```

### **🔧 エラーメッセージ別対処法**

| エラーメッセージ | 対処法 |
|------------------|--------|
| `failed to synthesize instance` | 必要なtypeclassインスタンスを明示 |
| `type mismatch` | 型注釈を追加、引数の型を確認 |
| `application type mismatch` | 関数適用の引数順序と型を確認 |
| `pattern matching failed` | 引数の条件（微分可能性など）を確認 |
