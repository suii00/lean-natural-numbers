## 次の課題：線形関数の微分と接線

### 問題文（高校生向け）

線形関数 f(x) = ax + b の微分が定数 a になることを証明し、さらにこの結果を使って点 (x₀, f(x₀)) における接線の方程式を導出しましょう。これは「一次関数の傾きは一定」という高校の知識をLeanで厳密化する課題です。

### 数学的背景
- **高校での学習内容**：数学Ⅱ「微分法」における一次関数の微分と接線の方程式
- **Leanでの表現**：線形関数の微分可能性と導関数の値、そして接線の定義

### Lean定義

```lean
-- 必要なimport文
import Mathlib.Analysis.Calculus.Deriv.Basic
import Mathlib.Analysis.Calculus.Deriv.Add
import Mathlib.Analysis.Calculus.Deriv.Mul

-- 課題1: 線形関数の微分
theorem linear_deriv (a b : ℝ) :
  ∀ x : ℝ, deriv (fun x : ℝ => a * x + b) x = a := by
  sorry

-- 課題2: 線形関数は全域で微分可能
theorem linear_differentiable (a b : ℝ) :
  Differentiable ℝ (fun x : ℝ => a * x + b) := by
  sorry

-- 課題3: 接線の方程式の係数を求める
theorem tangent_line_linear (a b x₀ : ℝ) :
  let f := fun x : ℝ => a * x + b
  let tangent_slope := deriv f x₀
  let tangent_intercept := f x₀ - tangent_slope * x₀
  tangent_slope = a ∧ tangent_intercept = b := by
  sorry
```

### 証明のヒント

1. **線形関数の分解**：`a * x + b` を `(fun x => a * x)` と `(fun x => b)` の和として扱う
2. **微分の線形性**：`deriv_add` と `deriv_mul_const` を組み合わせる
3. **接線の考え方**：微分係数が接線の傾きであることを利用

### 大学レベルへの発展

- **アフィン変換**：より一般的な線形変換 T(x) = Ax + b（Aは行列）への拡張
- **微分の線形性**：微分作用素Dの線形性 D(af + bg) = aD(f) + bD(g) の一般理論
- **接線と線形近似**：Taylor展開の1次近似としての接線の理解
