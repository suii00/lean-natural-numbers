## 📋 補題分割戦略: 7段階構築

前回の経験から、複雑な定理は戦略的分割が不可欠です：

### **段階1**: ガロア拡大の基本構造
```lean4
-- 目標: K/F がガロア拡大であることの特徴付け
lemma galois_extension_characterization (K F : Type*) [Field K] [Field F] 
  [Algebra F K] : IsGalois F K ↔ IsSeparable F K ∧ IsNormal F K
```

### **段階2**: ガロア群の作用理論
```lean4
-- 目標: ガロア群が中間体に自然に作用することを確立
def galois_action (K F : Type*) [Field K] [Field F] [Algebra F K] :
  K ≃ₐ[F] K →* Equiv.Perm (IntermediateField F K)
```

### **段階3**: 基本対応の存在性
```lean4
-- 目標: 中間体 → ガロア群の部分群 の写像を構築
def intermediate_to_subgroup (K F : Type*) [Field K] [Field F] 
  [Algebra F K] (L : IntermediateField F K) : Subgroup (K ≃ₐ[F] K)
```

### **段階4**: 逆対応の存在性
```lean4
-- 目標: ガロア群の部分群 → 中間体 の写像を構築
def subgroup_to_intermediate (K F : Type*) [Field K] [Field F] 
  [Algebra F K] (H : Subgroup (K ≃ₐ[F] K)) : IntermediateField F K
```

### **段階5**: 対応の単射性
```lean4
lemma galois_correspondence_injective_left : Function.Injective intermediate_to_subgroup
lemma galois_correspondence_injective_right : Function.Injective subgroup_to_intermediate
```

### **段階6**: 対応の全射性と逆対応性質
```lean4
lemma galois_correspondence_bijective : 
  intermediate_to_subgroup.comp subgroup_to_intermediate = id
```

### **段階7**: 統合定理（中間体定理）
```lean4
theorem fundamental_theorem_of_galois_theory (K F : Type*) [Field K] [Field F] 
  [Algebra F K] [IsGalois F K] :
  ∃ f : IntermediateField F K ≃ Subgroup (K ≃ₐ[F] K), 
    ∀ L H, f L = intermediate_to_subgroup L ∧ f⁻¹ H = subgroup_to_intermediate H
```
