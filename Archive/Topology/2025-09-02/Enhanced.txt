TopPair と積・普遍性

TopPair.Hom は連続写像の束。TopPair.pair f g : Hom Z (X ⨯ Y) と proj₁_pair, proj₂_pair。
TopPair.product_universal_property f g : Continuous (λ z, (f z, g z)) ↔ Continuous f ∧ Continuous g.
C(X,Y) と評価・カリー化

abbrev TopPair.C X Y := TopPair.Hom X Y
instance instTopC : TopologicalSpace (C X Y) は (X → Y) の積位相からの誘導位相。
TopPair.eval X Y : C(X,Y) × X → Y と TopPair.continuous_eval X Y.
TopPair.curry (f : C(X⨯Y, Z)) : C(X, C(Y, Z)) と TopPair.uncurry (F : C(X, C(Y, Z))) : C(X⨯Y, Z) はどちらも連続（誘導位相の普遍性と continuous_pi を利用）。

商（quotient）での連続性一撃:
一般の商位相は TopologicalSpace.coinduced で構成できます。群の正規部分群による商（QuotientGroup）に合わせて TopGrp.Pair 版の「自然射の連続性」「普遍性」補題を追加可能です。
Exponential law の Homeomorph まで:
いまは C(X,Y) に「誘導位相（関数全体空間の部分位相）」を採用。指数法則の同相（Homeomorph (C(X×Y, Z)) (C(X, C(Y, Z)))）まで出すには、通常 Y の局所コンパクト性などの条件下で「コンパクト開位相」による C(X, Y) を使うのが定石です。必要な import と位相選定を分岐し、Homeomorph を提供する形に拡張可能です。
