This chapter is the \textbf{T2 documentation skeleton} for WBS item \textbf{A3 (five examples)}.
Each example instantiates the T1 interface
\texttt{ST.Ranked} and (when applicable) the bridge to \texttt{StructureTowerWithMin}.

\section{Scope and conventions}
\begin{itemize}
\item \textbf{Source of truth (Lean).} Each section points to a Lean file (module path) that
  defines the instance and the supporting lemmas.
\item \textbf{T2 goal.} Provide a stable narrative around:
  (i) the rank function, (ii) the induced layers, (iii) representative lemmas, and
  (iv) small executable demonstrations when feasible.
\item \textbf{Bonus isolation.} Lemmas with heavy Mathlib dependencies are moved to \emph{Bonus}
  files and are not used by the T2 main text.
\end{itemize}
