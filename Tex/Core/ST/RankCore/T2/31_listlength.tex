\subsection{Definition}
Fix a type $\alpha$. Consider $X := \mathrm{List}(\alpha)$ and index set $\mathbb{N}$.
\begin{definition}[List length rank]
Define $\rank(l) := \mathrm{length}(l)$.
Equivalently, the $n$-th layer is
\[
  \layer(n) := \{\, l : \mathrm{List}(\alpha) \mid \mathrm{length}(l) \le n \,\}.
\]
\end{definition}

\item \textbf{Monotonicity:} \texttt{list\_layer\_mono}.

\end{itemize}

\subsection{A RankHomLe arrow: List \texorpdfstring{$\to$}{->} Finset}
Assuming \texttt{[DecidableEq α]}, the map \texttt{List.toFinset} drops duplicates, hence decreases
cardinality and yields a \texttt{RankHomLe}:
\[
  \map(l) := l.\mathrm{toFinset},\qquad \indexMap := \mathrm{id}_{\mathbb{N}},\qquad
  \rank(\map(l)) \le \rank(l).
\]
In Lean: \texttt{listToFinsetRankHomLe}. This provides a canonical morphism connecting two A3 examples.
