### 2. 証明戦略

#### アプローチ1: 固有値の場合の直接証明
- 固有ベクトルvが存在する場合（Tv = μv）
- 内積の自己随伴性を使用：⟪Tv, v⟫ = ⟪v, Tv⟫
- μ⟪v,v⟫ = conj(μ)⟪v,v⟫ を導出
- ⟪v,v⟫ ≠ 0 なので μ = conj(μ)
- これは μ.im ≠ 0 と矛盾

#### アプローチ2: 近似固有値の場合
- スペクトルの点は近似固有値
- 任意のε > 0に対して‖(μ - T)v‖ ≤ ε‖v‖となるv ≠ 0が存在
- 内積を評価：|⟪(μ - T)v, v⟫| ≤ ‖(μ - T)v‖ * ‖v‖
- ⟪Tv, v⟫は実数（自己随伴性）
- |μ.im * ‖v‖²| ≤ ε‖v‖²
- εを任意に小さくできるので、μ.im = 0
