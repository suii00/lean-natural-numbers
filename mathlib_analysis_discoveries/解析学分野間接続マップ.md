## 🌊 主要な接続パターン

### 1. 複素解析 ↔ 実解析の深い関係

#### リウヴィルの定理の統一的視点
```
複素平面での有界性 ────────────┐
                          ↓
実軸での定数性 ←───── 解析関数の剛性
                          ↑
                   コーシー積分公式
```

**発見**: 一次元の条件（有界性）が全体の構造を完全に決定する驚異的な剛性

#### 成長条件と正則性
```
Little-o成長: (f-c) =o[(z-c)⁻¹] ──→ 除去可能特異点
             ↕
Big-O成長: f =O[rpow] ──→ メリン変換の正則性
```

### 2. フーリエ解析の中心的役割

#### 自己随伴性による統一
```
実フーリエ変換 ────── 𝓕(𝓕f) = f̌ ──────┐
      ↕                                ↓
複素フーリエ変換 ──── ⟪𝓕f,g⟫ = ⟪f,𝓕g⟫ ── 完全対称性
      ↕                                ↑
抽象フーリエ変換 ──── 局所凸空間での一般化 ──┘
```

**美しさ**: 対称性が異なる抽象化レベルで保持される

#### 測度論との深い関係
```
ルベーグ積分 ←──→ フーリエ変換 ←──→ プランシュレル定理
     ↕              ↕              ↕
L²空間の構造 ←──→ 等長変換 ←──→ 内積の保持
```

### 3. スペクトル理論の階層構造

#### 有限次元から無限次元への拡張
```
行列の対角化 ────────────┐
              ↓ 無限次元化
自己随伴作用素 ←── スペクトル定理 ── 直和分解
              ↕              ↕
            実固有値 ←──→ 正規直交基底
```

**統一原理**: 内積構造が対角化可能性を保証

### 4. 漸近解析の普遍的パターン

#### 成長の階層
```
多項式成長 ──→ 指数成長 ──→ 超指数成長
     ↕           ↕           ↕
シュワルツ空間 → L²空間 → 超関数空間
```

#### スターリング公式の美しい接続
```

        ↓                   ↓
```
