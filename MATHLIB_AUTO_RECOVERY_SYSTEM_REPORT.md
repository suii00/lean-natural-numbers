## 🏗️ システム構成

### コアコンポーネント (5システム)

#### 1. 自動復旧システム (`AutoRecoverySystem.ps1`)
- **機能**: エラー検出・分析・ロールバック・修正提案
- **特徴**:
  - 4種類のエラータイプ自動分類 (BuildFailure, ImportError, DependencyError, SyntaxError)
  - 重要度別エラー処理 (High/Medium/Low)
  - 自動修正提案システム
  - セーフティブランチ自動作成

#### 2. 問題分離フレームワーク (`ProblemIsolation.ps1`)
- **機能**: 問題コンポーネントの特定・分離・バックアップ
- **特徴**:
  - 3段階分離戦略 (FileLevel, ModuleLevel, ImportLevel)
  - インテリジェント相関分析
  - 自動バックアップ作成
  - 復元手順の自動生成

#### 3. 段階的再試行システム (`StagedRetrySystem.ps1`)
- **機能**: プログレッシブ復旧戦略実行
- **特徴**:
  - 5段階復旧ステージ (Minimal → Basic → Standard → Full → Complete)
  - 3つの戦略 (Progressive, Conservative, Aggressive)
  - 事前条件チェック・事後検証システム
  - 動的タイムアウト調整

#### 4. ブランチ管理システム (`BranchManager.ps1`)
- **機能**: Git ブランチの自動作成・管理・クリーンアップ
- **特徴**:
  - 5種類のブランチタイプ (safety, recovery, isolation, test, milestone)
  - 自動ネーミング規則
  - マージ戦略選択 (safe, recovery, squash)
  - 保持期間管理・自動クリーンアップ

#### 5. 復旧オーケストレーター (`RecoveryOrchestrator.ps1`)
- **機能**: 全システムの統合制御
- **特徴**:
  - 7フェーズ復旧プロセス実行
  - リアルタイム進捗通知
  - 包括的レポート生成
  - エラー履歴管理

### サポートツール

#### 6. クイックスタートシステム (`QuickStart.ps1`)
- **機能**: ユーザーフレンドリーなインターフェース
- **特徴**:
  - インタラクティブメニュー
  - ワンクリック復旧実行
  - システム状態表示
  - 組み込みヘルプシステム

## 🔧 技術的実装詳細

### エラー検出エンジン
```powershell
# エラーパターンマッチング例
$errorPatterns = @{
    "BuildFailure" = @{
        Patterns = @(
            "object file.*does not exist",
            "lean.*failed with code",
            "lake build.*failed"
        )
        Severity = "High"
        RecoveryStrategy = "Rollback"
    }
}
```

### 段階的復旧ロジック
```powershell
# 復旧ステージ構成例
$stages = @("Minimal", "Basic", "Standard", "Full", "Complete")

# ステージ別コマンド設定
"build" = @{
    MinimalCommand = "lake build MyProject"
    BasicCommand = "lake build"  
    StandardCommand = "lake build && lake test"
    FullCommand = "lake build && lake exe cache get && lake test"
    CompleteCommand = "lake clean && lake build && lake exe cache get && lake test"
}
```

### ブランチ管理システム
```powershell
# ブランチタイプ設定
$branchConfig = @{
    BranchTypes = @{
        "safety" = @{
            Prefix = "safety"
            Retention = 7  # days
            AutoCleanup = $false
        }
        "recovery" = @{
            Prefix = "recovery"
            Retention = 3  # days
            AutoCleanup = $true
        }
    }
}
```
