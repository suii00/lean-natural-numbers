# 期待値の線形性 (Linearity of Expectation) 証明ログ

## 開始: 2025-08-08

### 証明する定理
1. **スカラー倍**: E[cX] = cE[X]
2. **加法性**: E[X + Y] = E[X] + E[Y]  
3. **一般線形性**: E[aX + bY] = aE[X] + bE[Y]
4. **有限和**: E[Σᵢ aᵢXᵢ] = Σᵢ aᵢE[Xᵢ]
